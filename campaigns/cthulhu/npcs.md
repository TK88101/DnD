# 克蘇魯的呼喚 — NPC

> DM 使用此文件管理非玩家角色。每個 NPC 都有其秘密——有些是盟友，有些是敵人，有些……兩者皆是。

---

## 1. 亨利·阿米蒂奇教授

*「有些書不是用來閱讀的。它們是用來鎖起來的。」*

### 基本資料

| 項目 | 內容 |
|------|------|
| 身份 | 米斯卡塔尼克大學古代語言學教授 / 圖書館特藏室管理員 |
| 年齡 | 62 歲 |
| 外貌 | 銀白頭髮、深陷的眼窩、戴圓框眼鏡、總是穿著皺巴巴的西裝外套 |
| 性格 | 博學、謹慎、偏執、對學生和晚輩有父親般的關懷 |
| 位置 | 第一章：失蹤（被囚於印斯茅斯）/ 第二章後：米斯卡塔尼克大學 |

### 故事角色

阿米蒂奇教授是主線的引發者。他多年來一直在研究新英格蘭地區的超自然歷史，特別是印斯茅斯和鄧威治的異常現象。他發現了星之智慧教派和大袞教團之間的聯繫，並試圖收集足夠的證據揭露他們——但他被發現了。

### 對話風格
```
「孩子，你要明白——我不是瘋了。那些書裡寫的東西，
不是隱喻，不是寓言。它們是真實的記錄。
那些……東西……它們一直都在。只是我們選擇不去看。」
```

### DM 資訊（玩家不可見）

- 教授知道星之智慧教派在阿卡姆的核心成員名單（但因為囚禁而無法直接告知）
- 他在禁書室藏了一本《大袞之書》的翻譯手稿
- 他是銀色暮光會的秘密成員
- **弱點**：過度保護玩家，可能隱瞞危險情報以「保護」他們

### 互動能力

| 服務 | 效果 | 條件 |
|------|------|------|
| 翻譯古籍 | 翻譯任何已知古代語言的文本 | 需將文本帶到他面前 |
| 研究協助 | 提供 1 條特定主題的線索 | 每天限 1 次 |
| 禁書室鑰匙 | 提供禁書室的使用許可 | 需完成第二章任務 |

## 2. 威廉·巴頓警長

*「年輕人，我管這座鎮子二十年了。什麼怪事我沒見過？但我建議你——別多管閒事。」*

### 基本資料

| 項目 | 內容 |
|------|------|
| 身份 | 阿卡姆鎮警察局長 |
| 年齡 | 55 歲 |
| 外貌 | 粗壯、紅鼻子、厚重的灰色八字鬍、總是叼著雪茄、穿著深藍色警服 |
| 性格 | 固執、實際、對超自然嗤之以鼻、內心有正義感但被官僚主義壓制 |
| 位置 | 阿卡姆警察局 |

### 故事角色

巴頓警長代表「官方力量」的侷限性。他不相信超自然，會對玩家的調查設置阻礙（但不是出於惡意）。然而，當證據足夠充分時，他會成為強力的盟友。

### 對話風格
```
「失蹤案？我已經派人查過了。阿米蒂奇那老頭大概是
跑去歐洲什麼圖書館找他的爛書去了。你們大學的人
不都這樣嗎？」

*吐出一口煙霧*

「聽著，如果你有什麼具體的線索——我說的是真正的
線索，不是什麼鬼故事——你可以來找我。否則，
別浪費我的時間。」
```

### DM 資訊（玩家不可見）

- 巴頓警長的副手克萊頓是星之智慧教派的成員——巴頓不知道這件事
- 他的女兒瑪莎三年前精神失常，被送到州立精神病院（她其實是看到了深潛者）
- 如果玩家能提供足夠的物證（至少 3 件），他會動用警力協助
- **轉折**：在第三章，當他發現副手的背叛後，他會全力支持玩家

### 互動能力

| 服務 | 效果 | 條件 |
|------|------|------|
| 查閱案件檔案 | 獲取阿卡姆警方的案件記錄 | CHA DC 14（或提供有效證據降為 DC 10） |
| 請求警力支援 | 派遣 2 名警員協助 | 需提供 3 件以上物證，且在第三章後 |
| 簽發搜查令 | 合法搜查指定地點 | 需提供充分理由（CHA DC 16） |

## 3. 瑪格麗特·「瑪姬」·穆雷

*「死者有話要說。大多數人只是不願意聽。」*

### 基本資料

| 項目 | 內容 |
|------|------|
| 身份 | 阿卡姆古董店「暮光珍藏」店主 / 銀色暮光會成員 |
| 年齡 | 45 歲 |
| 外貌 | 黑色長髮、蒼白的面容、銳利的灰色眼睛、總是穿深紫色衣物、脖子上掛著銀質五芒星 |
| 性格 | 神秘、直率、冷幽默、對超自然知識如飢似渴、外冷內熱 |
| 位置 | 阿卡姆鎮 · 暮光珍藏古董店 |

### 故事角色

瑪姬是玩家在超自然領域最重要的導師和盟友。她經營的古董店是銀色暮光會在阿卡姆的據點。她可以鑑定超自然物品、提供防護儀式的材料，以及分享她多年研究的超自然知識。

### 對話風格
```
*她從書架後面抬起頭，用那雙灰色的眼睛打量你*

「你看到了什麼東西。不用說，我看得出來。
眼神會變——看過那些東西之後，人的眼神都會變。」

*她從玻璃櫃中取出一個銀色的護身符*

「戴上這個。不能保護你的命，但至少能保護你的腦子
多撐一會兒。」
```

### DM 資訊（玩家不可見）

- 她的丈夫十年前在調查印斯茅斯時失蹤——這是她加入暮光會的原因
- 她擁有一份不完整的星之智慧教派成員名單
- 她可以執行小型的防護和驅邪儀式
- **秘密**：她的丈夫沒有死。他被深潛者俘虜並被迫與深潛者混血。她不知道這件事。

### 互動能力

| 服務 | 效果 | 條件 |
|------|------|------|
| 鑑定超自然物品 | 鑑定任何超自然物品的功能和來歷 | 免費 |
| 出售防護物品 | 護身符、黑蠟燭、聖水等 | 按物品表價格 |
| 儀式協助 | 協助執行需要多人參與的封印或驅邪儀式 | 需提前告知 |
| 暮光會情報 | 分享暮光會收集的情報 | 需加入暮光會或獲得其信任 |

## 4. 賽拉斯·馬什

*「外來人。印斯茅斯不歡迎外來人。你最好趁還能走的時候離開。」*

### 基本資料

| 項目 | 內容 |
|------|------|
| 身份 | 印斯茅斯鎮大袞教團的大祭司 |
| 年齡 | 外表約 70 歲，實際年齡不明 |
| 外貌 | 矮胖、禿頂、凸出的蒼白眼球、皮膚灰白如魚腹、手指間有退化的蹼、說話時隱約可見頸部的鰓裂 |
| 性格 | 陰險、固執、對深潛者的信仰狂熱、對外來者充滿敵意 |
| 位置 | 印斯茅斯 · 大袞教團教堂 |

### 故事角色

馬什是大袞教團在印斯茅斯的領袖，也是印斯茅斯深潛者混血居民中「變異」最深的人之一。他組織了對阿米蒂奇教授的綁架，並正在籌備一場大型召喚儀式。他是第二章的主要反派。

### 對話風格
```
*他的嘴唇幾乎不動，聲音像從喉嚨深處擠出的氣泡*

「你們這些……陸地上的人。你們以為自己是萬物之靈。
可笑。在大袞面前，你們不過是……海浪上的泡沫。」

*他的眼球緩慢地轉動，凝視你*

「大袞的祝福即將降臨。擋路者……會成為獻祭。」
```

### DM 資訊（玩家不可見）

- 他已經完成了大部分的混血變異，再過幾年就會完全變成深潛者
- 他知道魔鬼礁下深潛者城市的入口
- 他隨身攜帶的金色印記（大袞之印）可以控制低等的深潛者
- **弱點**：極度恐光，強光會使他暫時失明（攻擊劣勢 2 回合）

### 戰鬥數據

| 屬性 | 值 |
|------|---|
| HP | 45 |
| AC | 13（魚鱗皮膚） |
| 攻擊 | 匕首 1d4+2 / 利爪 1d6+2 / 大袞之印（召喚 2 隻深潛者援軍） |
| 特殊 | 水中呼吸、暗視 60 呎、半水生（水中 AC +2）|

## 5. 伊麗莎白·克萊恩博士

*「精神病學教會了我一件事——有時候，瘋子看到的東西比正常人多。」*

### 基本資料

| 項目 | 內容 |
|------|------|
| 身份 | 阿卡姆聖瑪麗醫院精神科主任 |
| 年齡 | 38 歲 |
| 外貌 | 短棕色頭髮、嚴肅但溫和的面容、戴無框眼鏡、總是穿白色長袍、口袋裡永遠有一支鋼筆 |
| 性格 | 理性、同情、堅定、在科學和超自然之間掙扎 |
| 位置 | 阿卡姆 · 聖瑪麗醫院精神科 |

### 故事角色

克萊恩博士是阿卡姆為數不多能幫助角色恢復 SAN 的人。她的精神科病房裡住著幾名因「看到不該看的東西」而發瘋的患者——他們的瘋言瘋語中藏著重要的線索。

### 對話風格
```
*她摘下眼鏡擦了擦，疲憊的眼睛裡有一絲溫柔*

「三號病房的威爾金斯先生又開始畫那些圖了。
一模一樣的圖案，每次都是——像是章魚？不，
比章魚複雜得多。而且他在畫之前從來沒有見過
類似的東西。」

*她看著你，表情變得認真*

「我是一個科學家。我不相信超自然。但我也沒辦法
解釋病房裡正在發生的事情。如果你有答案，我願意聽。」
```

### DM 資訊（玩家不可見）

- 她的病人中有一名曾是星之智慧教派的逃脫者，知道教派的集會地點
- 她保存著過去十年所有「異常」病例的詳細記錄——這些記錄是重要線索
- 她對超自然的態度會隨劇情發展改變：從懷疑到震驚到接受
- **可能的轉折**：如果玩家分享了足夠的證據，她會主動提供她收集到的病人資料

### 互動能力

| 服務 | 效果 | 條件 |
|------|------|------|
| 心理治療 | 恢復 2d6 SAN | 50g，每週限 1 次 |
| 查閱病歷 | 獲取精神病患的記錄（可能含線索） | CHA DC 14 或提供合理理由 |
| 藥物提供 | 提供鎮定劑和其他精神科藥物 | 需醫療理由 |
| 專業意見 | 對超自然現象提供醫學角度的分析 | 免費 |

## 6. 奧利佛·克勞利

*「我已經見過了門的另一邊。我選擇不再害怕。你呢？」*

### 基本資料

| 項目 | 內容 |
|------|------|
| 身份 | 前米斯卡塔尼克大學考古學教授 / 星之智慧教派的二把手 |
| 年齡 | 50 歲 |
| 外貌 | 瘦高、灰色的眼睛有不自然的光芒、修剪整齊的黑色鬍子、穿著考究的三件套西裝、右手無名指戴著一枚刻有星形圖案的黑石戒指 |
| 性格 | 優雅、博學、危險、對舊日支配者有近乎宗教性的崇拜、對「啟蒙」的執念 |
| 位置 | 阿卡姆上流社會 / 星之智慧教派秘密據點 |

### 故事角色

克勞利是本戰役最危險的反派之一。他曾是受人尊敬的學者，但在一次中東考古發掘中「接觸」到了某些東西，從此墮入了星之智慧教派。他是教派在阿卡姆的實際執行者，負責滲透上流社會和大學。

### 對話風格
```
*他優雅地轉動手中的雪利酒杯*

「你知道嗎？人類最大的缺陷不是無知——是恐懼。
我們恐懼未知，恐懼黑暗，恐懼那些比我們更偉大的存在。
但如果你能克服恐懼……」

*他微笑，但笑容沒有到達那雙灰色的眼睛*

「……你就能看到真正的世界。美麗得令人窒息。
字面意義上的。」
```

### DM 資訊（玩家不可見）

- 他知道星之智慧教派的最終計劃——利用三角陣法打開裂隙
- 他在阿卡姆擁有大量的政治和經濟影響力，可以操縱警方和媒體
- 他的黑石戒指是一個超自然法器，可以短暫召喚暗影獵犬
- **弱點**：他的人性並未完全消失。如果玩家能揭露教派的真正目的（犧牲全城人命），他可能會動搖
- **戰鬥前選項**：如果玩家有足夠的證據和說服力（CHA DC 20），可以嘗試策反他

### 戰鬥數據

| 屬性 | 值 |
|------|---|
| HP | 55 |
| AC | 14（超自然護盾） |
| 攻擊 | 小型手槍 1d8 / 黑石戒指（召喚暗影獵犬 1 隻，每場限 1 次） / 精神衝擊（WIS DC 15，失敗損失 2d6 SAN） |
| 特殊 | 黑暗視覺、精神抗性（SAN 損失免疫）、超自然護盾（每天 3 次，一次攻擊傷害減半） |
