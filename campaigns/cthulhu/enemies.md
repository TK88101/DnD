# 克蘇魯的呼喚 — 敵人

> 本文件列出戰役中可能遭遇的所有敵人。DM 根據玩家等級和場景選擇合適的敵人。
> 記住：在克蘇魯世界中，最可怕的敵人不是最強的——而是最不可理解的。

---

## 普通敵人

### 邪教徒（Lv1-20，貫穿全篇）

*「伊阿！伊阿！克蘇魯·法坦！」——邪教徒的詠唱，在黑暗中回響*

邪教徒是各種邪教組織的人類成員。他們外表與常人無異，這正是他們危險的原因——你的鄰居、你的同事、你的醫生，都有可能在月圓之夜戴上面具。

## 超自然敵人

### 深潛者（Lv8-15）

*它從水中升起。六呎高的身軀覆蓋著灰綠色的鱗片，巨大的蛙眼在月光下反射出冰冷的光芒。它張開佈滿尖牙的嘴，發出的聲音像是金屬在水中刮擦。*

古老的水陸兩棲種族，自稱為大袞的子民。他們在大洋深處建立了宏偉的城市，並世代與印斯茅斯的人類通婚。

### 飛水母 / 拜亞基（Lv10-16）

*黑色的天空中出現了一個不可能存在的輪廓。它像蝙蝠、像昆蟲、又像某種腐爛的水母。翅膀拍動時沒有聲音，但空氣中充滿了一種讓你想吐的甜膩味道。*

星際間的生物，能在真空和大氣中飛行。邪教徒通過特殊儀式召喚它們作為坐騎或戰鬥使魔。

| 屬性 | 值 |
|------|---|
| HP | 50 |
| AC | 15（奇異甲殼） |
| STR 16, DEX 18, CON 14, INT 6, WIS 12, CHA 3 |
| 攻擊 | 利爪 1d8+3（2 次）/ 撕咬 1d10+3 / 纏繞觸手（STR DC 15，失敗被抓住並帶到空中，每回合 1d6 擠壓傷害） |
| 特殊 | 飛行速度 60 呎、暗視 120 呎、免疫冷凍和真空傷害、空間跳躍（每場限 1 次，瞬間移動到 60 呎內任意位置） |
| 弱點 | 光明和火焰造成 +50% 傷害、墜落時受到正常墜落傷害 |
| SAN 損失 | 首次遭遇 -2d6 |
| 掉落 | 奇異甲殼碎片（煉金材料，50g）、體內可能有被吞噬者的遺物（線索） |

### 暗影獵犬（Lv12-18）

*你沒有看到它——你感覺到它。房間角落的陰影突然變得太深了。空氣中傳來低沉的呼嚕聲，像狗，但比任何狗都更……錯誤。然後你聞到了那股氣味。臭氧和腐肉的混合。*

存在於空間夾縫中的掠食者，被邪教徒用特定儀式和法器召喚。它們不完全存在於物質世界，這使它們極難對付。

| 屬性 | 值 |
|------|---|
| HP | 65 |
| AC | 16（半實體化——物理攻擊 50% 概率穿過） |
| STR 18, DEX 16, CON 14, INT 8, WIS 14, CHA 4 |
| 攻擊 | 暗影撕咬 2d8+4（無視護甲）/ 暗影衝刺（直線 30 呎，路徑上所有目標 2d6 傷害 + WIS DC 15，失敗恐懼 2 回合）/ 空間裂擊（對 15 呎範圍所有目標 3d6 傷害）|
| 特殊 | 半實體化（物理攻擊 50% 穿透）、暗視 120 呎、空間穿行（可穿過牆壁）、恐懼光環（30 呎內，每回合 WIS DC 13 或 -1d4 SAN） |

| SAN 損失 | 首次遭遇 -2d8 |
| 掉落 | 暗影碎片（極稀有材料，可用於製作銀質武器或防護符文） |

### 修格斯（Lv15-20）

*那不是任何已知的生命形式。它是一團巨大的、不斷變形的黑色物質，表面此起彼伏地冒出眼睛、嘴巴和觸手，然後又消融回那團恐怖的肉泥中。它移動時發出令人作嘔的濕潤聲，身後留下冒著泡的黏液痕跡。*

遠古奴僕種族，原本由「古老者」創造的生物工具。它們幾乎不可被摧毀，只能被驅趕或逃離。

| 屬性 | 值 |
|------|---|
| HP | 150 |
| AC | 18（不斷變形的表面） |
| STR 24, DEX 8, CON 22, INT 4, WIS 8, CHA 1 |
| 攻擊 | 偽足碾壓 3d10+7 / 吞噬（STR DC 20，失敗被吞入體內，每回合 4d6 酸蝕傷害）/ 觸手連擊（攻擊 30 呎內所有目標各 2d8+4） |
| 特殊 | 再生（每回合恢復 15 HP）、酸蝕身體（近戰攻擊者受 1d6 酸蝕反傷）、免疫毒素/疾病/精神影響、變形（可壓縮通過任何寬度 1 呎以上的縫隙） |
| 弱點 | 火焰阻止再生 2 回合、電擊傷害 +100%、強光使其暫時退縮（WIS DC 12）|
| SAN 損失 | 首次遭遇 -2d8 |
| **DM 提示** | 修格斯不適合正面戰鬥。建議用它作為追逐場景或環境威脅。如果玩家堅持戰鬥，確保他們知道撤退是選項。 |

### 星之眷族（Lv16-20）

*它站在那裡——如果那算是「站」的話。三公尺高的桶狀身體，頂端是海星狀的頭部，五條觸手臂末端各有不同的器官。它的身體構造完全不符合任何地球生物學。但最令人不安的是，它看起來是……有智慧的。*

來自遙遠星球的古老種族，在地球文明誕生之前就已經造訪此地。他們不是邪惡的——但他們完全不把人類當回事。

| 屬性 | 值 |
|------|---|
| HP | 100 |
| AC | 18（外星甲殼） |
| STR 20, DEX 14, CON 18, INT 22, WIS 18, CHA 8 |
| 攻擊 | 觸手鞭擊 2d8+5（射程 15 呎，2 次/回合）/ 精神衝擊波（30 呎範圍，WIS DC 18，失敗 3d8 精神傷害 + 2d6 SAN 損失）/ 閃電裝置（50 呎直線，4d10 雷電傷害，DEX DC 16 減半） |
| 特殊 | 心靈感應（無需語言溝通）、飛行（緩慢）、再生（5 HP/回合）、多重行動（每回合 2 個動作）|
| 弱點 | 火焰傷害正常且阻止再生 1 回合 |
| SAN 損失 | 首次遭遇 -3d6 |
| 掉落 | 外星科技碎片（極稀有）、星圖（可能是關鍵線索） |
| **DM 提示** | 星之眷族可以被溝通。如果玩家嘗試（且有神秘學者使用心靈感應），可以進行非戰鬥互動。它們可能提供關於舊日支配者的情報——但其方式可能讓人發瘋。 |

## Boss 級敵人

### 舊日支配者的觸手（最終 Boss，Lv20）

*裂隙撕開了。*

*天空不再是天空——那是一片黑色的海洋，而「它」正從海洋中伸出。你看到的不是全貌，只是一小部分——一條觸手，如果那能被稱為觸手的話。它比教堂的鐘塔還粗，覆蓋著無數蠕動的吸盤，每個吸盤中心都有一隻半開半閉的眼睛。它的表面在物質和虛無之間閃爍，你的大腦尖叫著告訴你：這不可能存在。但它就在那裡。*

*你聽到了聲音——不是用耳朵，而是直接在腦子裡。那不是語言，不是音樂，不是任何你能理解的東西。但你知道了它的名字。你不想知道。但你已經知道了。*

**注意：玩家不會面對完整的舊日支配者。他們面對的是從裂隙中伸出的一條觸手——而這已經足以毀滅一切。**

| 屬性 | 值 |
|------|---|
| HP | 300（但觸手有 3 個弱點節點，各 100 HP，摧毀全部可將觸手逼退） |
| AC | 20 |
| STR 30, DEX 10, CON 28, INT -, WIS -, CHA - |
| 攻擊 | 觸手橫掃（所有 30 呎內目標，4d10+10 傷害，DEX DC 18 減半）/ 吸盤凝視（單體，WIS DC 20，失敗損失 3d10 SAN + 恐慌 2 回合）/ 虛空波動（全場，所有生物受 3d8 精神傷害，無豁免）/ 觸手壓碎（抓取一名目標，STR DC 22，失敗每回合 4d8 傷害） |
| 特殊 | 免疫所有精神效果、免疫毒素/疾病、抗性：所有物理傷害（減半）、再生（每回合 20 HP，除非弱點節點被攻擊）、恐懼光環（100 呎，每回合自動 -1d6 SAN） |
| 弱點節點 | 觸手上 3 個發光的節點（AC 14），對銀質武器和火焰無抗性。摧毀節點時觸手痙攣 1 回合（不行動），全部摧毀則觸手退回裂隙。 |
| 線索加成 | 收集 75% 以上線索的隊伍知道節點位置，攻擊節點 +2；100% 線索的隊伍擁有封印儀式，可在摧毀 2 個節點後直接封印裂隙 |
| SAN 損失 | 直視：-3d10（WIS DC 20 減半） |

### Boss 戰流程建議

**第一階段（3 個節點完好）：**
- 觸手全力攻擊，使用橫掃和壓碎
- 每 2 回合使用虛空波動
- 恐懼光環持續生效
- 小型觸手從裂隙中伸出騷擾（HP 20，AC 12，攻擊 1d8+4）

**第二階段（2 個節點被摧毀）：**
- 觸手開始使用吸盤凝視（單回合最強攻擊）
- 虛空波動頻率增加為每回合
- 裂隙開始擴大——如果不在 5 回合內摧毀最後一個節點，裂隙將不可逆轉地打開

**第三階段（觸手退回 / 封印儀式）：**
- 最後一個節點被摧毀，或封印儀式啟動
- 觸手開始退回裂隙，但會進行最後的瘋狂攻擊
- 玩家需要在觸手退回的同時封閉裂隙（需要神秘學者的封印儀式或特殊物品）
