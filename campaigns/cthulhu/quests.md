# 克蘇魯的呼喚 — 任務

> DM 使用此文件管理主線和支線任務。每個任務包含目標、線索、敵人和獎勵。
> 克蘇魯戰役的任務以調查和解謎為核心，戰鬥為輔助手段。

---

## 主線任務

### 第一章：消失的教授（Lv1-5）

---

#### 主線 1-1：教授的辦公室

| 項目 | 內容 |
|------|------|
| 觸發 | 戰役開始，玩家收到阿米蒂奇教授的訊息 |
| 目標 | 搜索教授的辦公室，找到他失蹤的線索 |
| 地點 | 米斯卡塔尼克大學 · 語言學系辦公室 |
| 建議等級 | 1 |

**任務流程：**

1. 玩家進入教授辦公室（門未鎖——或已被撬開）
2. 辦公室被翻過，但不像是警方搜索的方式
3. 搜索檢定（INT DC 10）：
   - 成功：找到教授桌子第三個抽屜裡的**筆記本**（初始線索）
   - 大成功（DC 15+）：額外發現地板上的泥土痕跡——不是普通的泥土，帶有鹹味（印斯茅斯的土壤）
4. 筆記本內容：教授記錄了他對《大袞之書》的翻譯進度，提到「印斯茅斯的馬什家族」和「鄧威治的韋特利家族」之間的聯繫
5. **可選：** 查閱教授的信件（INT DC 12）——發現一封來自鄧威治的信，署名「一個擔憂的鄰居」

**敵人：** 無（但離開時可能注意到有人在走廊盡頭觀察——星之智慧教派的監視者）

**獎勵：**
- 線索：阿米蒂奇的筆記本
- 經驗：50 XP
- 解鎖：主線 1-2 和 1-3

#### 主線 1-2：圖書館的秘密

| 項目 | 內容 |
|------|------|
| 觸發 | 完成主線 1-1，筆記本提到《大袞之書》 |
| 目標 | 在大學圖書館調查《大袞之書》和相關資料 |
| 地點 | 米斯卡塔尼克大學 · 奧恩圖書館 |
| 建議等級 | 1-2 |

**任務流程：**

1. 在公開藏書區搜索（INT + 圖書研究 DC 12）：找到關於印斯茅斯歷史的書籍——印斯茅斯曾是繁榮的港口，1846 年後突然衰敗，居民變得封閉且怪異
2. 嘗試進入禁書室（CHA DC 15 說服管理員，或 DEX DC 14 潛入）：
   - 成功：發現教授的借閱記錄——他借出了《大袞之書》的拉丁語殘本，但書從未歸還
   - 額外發現：《無名邪教》中關於「深潛者」的描述（閱讀：SAN -1d4）
3. 離開圖書館時，遭遇一名陌生人攔路——自稱是「教授的老朋友」，詢問教授的下落。心理學檢定（WIS DC 12）可感知他在說謊。
   - 如果揭穿：他匆匆離去（邪教徒偵察員）
   - 如果不揭穿：他會試圖誤導玩家（提供假線索）

**敵人：** 1 名低階邪教徒（可能發展為戰鬥或逃跑追逐）

**獎勵：**
- 線索：印斯茅斯歷史資料、禁書室借閱記錄
- 經驗：75 XP
- 物品：《無名邪教》節譯本（如果進入禁書室）

#### 主線 1-3：鄧威治之行

| 項目 | 內容 |
|------|------|
| 觸發 | 完成主線 1-1，筆記本提到鄧威治和韋特利家族 |
| 目標 | 前往鄧威治調查韋特利農莊 |
| 地點 | 鄧威治 · 韋特利農莊 |
| 建議等級 | 2-4 |

**任務流程：**

1. 前往鄧威治（途經森林，擲隨機遭遇表）
2. 到達鄧威治，鎮民態度冷淡。雜貨店老闆勉強願意交談（CHA DC 10）：
   - 「韋特利家？那個地方不吉利。老韋特利死後沒人去過。他的孫子威爾伯更是個怪物——不是說壞話，但他真的……不太像人。」
3. 韋特利農莊搜索：
   - 主屋（INT DC 12）：找到威爾伯的日記——記載了他與「海底之物」的聯繫和一個召喚儀式的日期
   - 穀倉（上鎖，DEX DC 14 開鎖或 STR DC 16 破門）：內部刻滿符文，中央有一個石祭壇，祭壇上有乾涸的血跡。SAN -1d4。
   - 地下室（隱藏入口，INT DC 15）：發現阿米蒂奇教授的一件個人物品（眼鏡盒）——他來過這裡
4. **遭遇：** 搜索穀倉時，2-3 名邪教徒出現（低階，他們是韋特利的追隨者）

**敵人：** 低階邪教徒 x2-3

**獎勵：**
- 線索：威爾伯日記、石祭壇符文拓本、教授曾到此的證據
- 經驗：150 XP
- 解鎖：主線 1-4

#### 主線 1-4：黑水沼澤的營地

| 項目 | 內容 |
|------|------|
| 觸發 | 完成主線 1-3，日記提到「沼澤中的集會」 |
| 目標 | 找到邪教徒在黑水沼澤的營地，獲取更多情報 |
| 地點 | 森林沼澤 · 黑水沼澤 |
| 建議等級 | 3-5 |

**任務流程：**

1. 穿越沼澤（WIS DC 12 找到路徑，失敗迷路 2 小時）
2. 發現邪教營地（DEX + 潛行 DC 13 不被發現）
3. 偵察營地：5 名邪教徒正在準備儀式。他們談論「印斯茅斯的準備工作已就緒」和「教授已被送走」
4. 玩家選擇：
   - **潛入**：DEX DC 14，可偷取儀式材料和文件
   - **攻擊**：與 5 名低階邪教徒戰鬥
   - **跟蹤**：儀式結束後跟蹤領頭者（WIS DC 13，引向印斯茅斯的方向）
5. 無論選擇什麼，最終獲得：教授被送往印斯茅斯馬什工廠的確認

**敵人：** 低階邪教徒 x5

**獎勵：**
- 線索：印斯茅斯是教授被關押的地點、儀式的目的是「迎接深海之主」
- 經驗：200 XP
- 物品：儀式匕首、邪教典籍
- 解鎖：第二章

### 第二章：印斯茅斯之影（Lv6-10）

---

#### 主線 2-1：抵達印斯茅斯

| 項目 | 內容 |
|------|------|
| 觸發 | 完成第一章 |
| 目標 | 潛入印斯茅斯，調查教授的關押地點 |
| 地點 | 印斯茅斯 |
| 建議等級 | 6-7 |

**任務流程：**

1. 抵達印斯茅斯。空氣中的魚腥味令人作嘔。街道空蕩蕩的，但你能感覺到窗簾後面有眼睛在盯著你。SAN -1d4（首次進入印斯茅斯）。
2. 吉爾曼旅館登記（唯一選擇）。老闆態度冷淡，房間潮濕。注意：門鎖不牢。
3. 調查鎮子：
   - 大袞教團教堂（外觀可疑，夜晚有詠唱聲）
   - 馬什工廠（有人守衛）
   - 碼頭（老漁夫可能願意交談——CHA DC 14）
4. **夜間事件：** 凌晨，旅館門被打開。2 名印斯茅斯混血居民試圖搜查你的房間。
   - 戰鬥或逃跑

**敵人：** 印斯茅斯混血居民 x2

**獎勵：**
- 線索：馬什工廠有地下設施、大袞教團的儀式時間表
- 經驗：250 XP

---

#### 主線 2-2：馬什工廠潛入

| 項目 | 內容 |
|------|------|
| 觸發 | 完成主線 2-1 |
| 目標 | 潛入馬什工廠，救出阿米蒂奇教授 |
| 地點 | 印斯茅斯 · 馬什煉金工廠 |
| 建議等級 | 7-9 |

**任務流程：**

1. 工廠有 2 個入口：正門（守衛 x4）和地下水道入口（偵查 DC 14 發現）
2. 地下水道充滿齊腰深的海水和可怕的氣味。SAN -1d4。
3. 工廠地下層：牢房區關押著教授和 2 名其他囚犯
4. 教授虛弱但活著。他告訴玩家：「儀式在明天滿月時進行。在魔鬼礁。你們必須阻止他們。」
5. 逃出工廠：戰鬥或潛行。如果觸發警報，追逐場景開始。

**敵人：** 中階邪教徒 x3、印斯茅斯混血居民 x4、深潛者步兵 x1（首次遭遇，SAN -2d6）

**獎勵：**
- NPC：阿米蒂奇教授歸隊（提供情報和翻譯服務）
- 線索：魔鬼礁的位置和儀式細節
- 經驗：400 XP
- 解鎖：主線 2-3

### 第三章：星之智慧（Lv11-15）

#### 主線 3-1：回到阿卡姆
- 回到阿卡姆後發現城鎮氣氛變了。一些熟悉的 NPC 行為怪異。
- 調查發現星之智慧教派已滲透警察局和市政府。
- 巴頓警長的副手被揭露為教派成員。
- 目標：揭露教派在阿卡姆的網絡。

#### 主線 3-2：暮光珍藏
- 瑪姬被教派追殺，需要保護她和她的店。
- 正式加入銀色暮光會，獲得暮光之星。
- 獲得教派成員的部分名單。

#### 主線 3-3：詭異莊園調查
- 追蹤教派領袖奧利佛·克勞利到城郊的莊園。
- 詳見副本 `dungeons/manor.md`。
