# 結雲村 NPC

> 結雲村人物典冊 | 適用規則：D&D 5e 怪物獵人改版

---

> **設定說明：** 怪物獵人世界中，獵人是獨立作業或與他人協力狩獵的專業人士。村中 NPC 並非戰鬥夥伴，而是提供各種後勤支援、情報、裝備鍛造的服務人員。真正在戰場上陪同獵人的是「隨從」（Palico 貓妖或 Palamute 犬妖），與人類村民不同。

---

## 一、里長 福永

**定位：** 村長 / 主線任務發布者

**性格：** 沉穩睿智，歷經滄桑卻不失溫暖。說話緩慢但字字有分量，偶爾會講述過去的艱辛歲月。對每位來到結雲村的獵人都視如子侄。

**背景：**
福永自幼在結雲村長大，年輕時也曾是名優秀的獵人，膝蓋受了重傷後卸下獵人身分接任村長一職。他見過無數怪物災害，也親手送走過許多優秀的獵人。正因如此，他比任何人都更清楚每一次委託的份量。

**台詞：**
- 「年輕人，結雲村能延續至今，靠的不是村牆，而是每一位願意走出去保護村子的獵人。你，準備好了嗎？」
- 「那座山最近不太平靜……老骨頭能感覺到地底的顫動。你去查查，小心為上。」
- 「捕獲不是失敗，那是一種智慧。知道什麼時候留手，跟知道什麼時候出手一樣重要。」
- 「每次出發前，先去喵太那裡吃頓飽飯。空著肚子上戰場，是對怪物的不尊重，也是對自己的不尊重。」

**服務內容：**
- 發布主線任務與支線任務
- 提供怪物出沒的地域情報（加值 +2 於相關知識檢定）
- 重大任務完成後給予特殊獎勵（珍稀素材或專屬裝備圖紙）
- 村莊升級委託（達成條件後解鎖新設施）

## 二、加工屋老闆 火山

**定位：** 鐵匠 / 武器裝備鍛造

**性格：** 豪爽直接，對手藝極度自豪，對於劣質素材或草率的鍛造委託會當場翻臉。外表粗獷但內心細膩，打造每件裝備前都會仔細詢問獵人的使用習慣。聲音洪亮，笑聲更洪亮。

**背景：**
火山出身鐵匠世家，祖父曾為傳說中的獵人鍛造過古龍級別的裝備。他接手加工屋後立誓「只打造最好的裝備」，對付出生命換來的怪物素材有著近乎神聖的尊重。他的秘密是：壁櫃深處藏著一片無法鑑定的鱗片，他推測那是黑龍的……

**台詞：**
- 「你帶來的素材……哈！這品質！好東西啊！讓我來，保你打出一把讓怪物膽寒的傢伙！」
- 「別急，好刀需要時間。你急什麼？你急著去死嗎？」
- 「這柄刀我用了三天三夜打出來。你最好好好愛惜它。弄壞了你別想我再接單。」
- 「古龍的素材……」（沉默片刻）「……你不知道拿著這東西站在我面前，有多少人會想搶走你吧。小心點，孩子。」

**服務內容：**

| 服務項目 | 說明 | 費用 |
|----------|------|------|
| 武器鍛造 | 以素材打造新武器（參照武器升級樹） | 見裝備表 |
| 武器強化 | 在現有武器上進行升級 | 見裝備表差額 |
| 防具鍛造 | 以怪物素材製作防具套裝 | 見裝備表 |
| 防具強化 | 強化現有防具的防禦值 | 原費 ×30% |
| 素材鑑定 | 鑑定不明怪物素材的品質與用途 | 5g/件 |
| 裝備修繕 | 修復損壞裝備（戰鬥中裝備耐久歸零後） | 原費 ×20% |

## 三、集會所受付嬢 日向

**定位：** 集會所受付嬢（大廳接待員）/ 上位任務與多人任務發布

**性格：** 開朗熱情，笑容永遠掛在臉上。擅長安慰因任務失敗而沮喪的獵人，也擅長在獵人太過自信時潑冷水。雖然外表輕快，但對於集會所管理事務十分嚴謹認真。

**背景：**
日向從小在集會所長大，母親是前任受付嬢，父親是知名獵人（現已退休）。她見過無數獵人從這道門走進來，也見過有些人再也沒有回來。正因如此，她在歡送獵人出發時，每次都會悄悄祈禱。

**台詞：**
- 「歡迎來到集會所！今天要接什麼任務呢？上位的怪物可不是鬧著玩的，確認準備好了再出發喔！」
- 「哎呀，失敗了也沒關係！失敗只是在說：這次的準備還不夠。下次做好萬全準備再來！」
- 「你知道嗎，最厲害的獵人從來不說『我不需要幫手』。集會所就是讓大家一起變強的地方！」
- 「（壓低聲音）最近上位的轟龍比以前頻繁很多……村長沒說，但我聽到了一些傳言。有空的話，問問龍人族的長老吧。」

**服務內容：**

| 服務項目 | 說明 |
|----------|------|
| 上位任務發布 | 難度 ★★★ 以上的怪物討伐、捕獲任務 |
| 多人協力任務 | 最多 4 位獵人共同參與（獎勵按人頭追加） |
| 探索任務 | 自由探索地圖採集素材、捕獲小型生物 |
| 集會所設施使用 | 鍛造素材箱、任務告示板、其他獵人交流 |
| 緊急任務通報 | 突發大型怪物入侵事件的即時通知 |

## 四、里受付嬢 皐月

**定位：** 村莊受付嬢 / 低級任務與村莊委託發布

**性格：** 溫柔沉穩，說話輕柔，與姐姐日向的熱情形成明顯對比。觀察力極敏銳，往往能看出獵人的疲憊或情緒問題，會在不動聲色的情況下給予關心。

**背景：**
皐月是日向的妹妹，從小體弱不能成為獵人，卻有著超群的記憶力和觀察力。她熟記結雲村近百年的所有怪物出沒紀錄，許多老獵人都說她「知道的比里長還多」。她默默喜歡著一位常年在外的獵人，但從不說出口。

**台詞：**
- 「歡迎回來……您看起來很疲憊。任務可以等，先休息吧。」
- 「這個委託是村民托付的，不是討伐，是保護。請您……多加小心。」
- 「（翻著厚重的紀錄冊）三十二年前，同樣的地點有一隻角龍出沒。那次，有兩位獵人沒有回來。這次……我相信您。」
- 「姐姐說話很大聲，但她其實比我更擔心每一位出任務的獵人。我們兩個都是。」

**服務內容：**

| 服務項目 | 說明 |
|----------|------|
| 村莊委託任務 | 難度 ★ 至 ★★★ 的討伐、採集、護衛任務 |
| 緊急村莊保護任務 | 怪物侵入村莊時的緊急應對任務 |
| 歷史紀錄查詢 | 查閱過去的怪物出沒紀錄（+2 於相關知識檢定） |
| 村民情報 | 村民目擊報告整理，提供目標位置線索 |

## 五、貓廚 喵太

**定位：** 隨從廚師（Felyne Chef）/ 出發前料理增益

**性格：** 活潑可愛，對料理有著近乎宗教式的熱情。廚房是他的聖地，任何人未經允許進入都會被他用鍋鏟趕出去——即使是里長。偶爾用喵語說話，讓人搞不清楚他到底在說什麼，但料理的美味讓人無法計較。

**背景：**
喵太據說是從遙遠的古龍地帶流浪而來，身上帶著一本破舊的食譜，那是他已故師父留下的。他用這本食譜在結雲村紮根，開始為獵人烹飪出發前的料理。他深信：「好的料理能改變狩獵的命運喵～」

**台詞：**
- 「今天的食材是用生命換來的喵！本大廚的料理也是用靈魂做出來的喵！快坐下，吃飯喵！」
- 「哎呀，那個素材組合……不不不，等一下，讓本大廚想想……喵！可以做出一種特別的料理！」
- 「（嚴肅地）空肚子出門的獵人，是對怪物失禮喵。怪物值得被有實力的獵人挑戰，不是嗎喵？」
- 「上次那位大獵人說我的料理救了他的命喵……（擦眼淚）……這就是為什麼本大廚每天早起的原因喵！」

**服務內容：**

每次出發前可選擇一份料理，效果持續至當次任務結束（或場景結束）。

| 攻擊力大餐 | 攻擊判定 +2 | 8g |
| 防禦力大餐 | AC +1 | 8g |
| 火焰特耐套餐 | 火焰傷害抗性（傷害減半） | 10g |

## 六、雜貨商 小狸

**定位：** 物品商人 / 消耗品販售

**性格：** 精明算計，但並非惡意，只是非常清楚自己的商品價值。喜歡和獵人聊天，消息靈通，有時會透露一些「額外情報」——當然，不是免費的。討價還價是他的樂趣，但他有自己的底線：從不賣假貨。

**背景：**
小狸祖籍不詳，據說年輕時曾是個走商，跑遍了各地的獵人村。他在結雲村定居是因為「這裡的怪物夠多，生意夠穩定」。他的倉庫被認為是「神秘空間」，永遠比看起來的大，裡面似乎有著無窮無盡的商品。

**台詞：**
- 「歡迎光臨！小狸的店，保真保品質，不滿意直接來找我，但我保證你會滿意的哈哈哈！」
- 「這個閃光彈……說真的，比你的劍還重要。相信我，我見過太多不用閃光彈的獵人，然後……」（做出抹脖子的動作）
- 「（壓低聲音）有消息說最近山的深處有不尋常的聲響……想聽嗎？消息費，10金幣。」
- 「捕獲用麻醉球？哦，你打算活捉？智慧啊，這才是獵人的最高境界！給你打個九折。」

**服務內容：**

販售所有消耗品（參照物品表一）。另外提供：

| 特殊服務 | 說明 | 費用 |
|----------|------|------|
| 情報購買 | 目標怪物弱點、習性、出沒地點 | 10-30g（依情報等級） |
| 稀有素材收購 | 收購獵人不需要的素材（售價約市值 40%） | — |
| 套購折扣 | 購買 5 件以上同類消耗品，九折優惠 | — |
| 預訂服務 | 預訂特殊物品，3 日後到貨 | 正常價格 +10% |

## 七、隨從管理員 （Buddy Handler）

**定位：** 隨從（Palico / Palamute）管理員 / 夥伴系統負責人

**姓名：** 渡

**性格：** 話不多，但對每一隻隨從都瞭若指掌。與其說是管理員，不如說是隨從們的父母。他能和貓妖與犬妖溝通——不是用語言，而是用某種只有他們才能理解的默契。對獵人有一點冷漠，但對隨從永遠溫柔。

**背景：**
渡年輕時是一位獵人，他的 Palico 夥伴「銀爪」在一次任務中為了救他而犧牲。此後他放下武器，轉而照顧每一隻來到結雲村的隨從，讓牠們都能找到值得牠們信任的主人。銀爪的項圈至今掛在他的頸上。

**台詞：**
- 「（凝視著你）……你值得信賴嗎？牠們可是把命交給你的。」
- 「Palico 不是工具，不是肉盾。牠是你的夥伴。記好了。」
- 「這隻……有點特別。牠挑主人。你去試試，如果牠願意跟你，那就是緣分。」
- 「（摸著頸上的項圈，輕聲）……好好的回來，帶著牠一起。」

**服務內容：**

| 服務項目 | 說明 | 費用 |
|----------|------|------|
| 招募 Palico | 貓妖夥伴，具備戰鬥輔助、採集、回復技能 | 100g（初次） |
| 招募 Palamute | 犬妖夥伴，提供騎乘移動、高機動戰鬥輔助 | 150g（初次） |
| 隨從訓練 | 提升隨從技能等級（需任務點數） | 50g/等級 |
| 隨從裝備 | 為隨從鍛造專屬裝備（提升防禦與技能） | 60-300g |
| 隨從任務派遣 | 在玩家外出時派遣隨從執行副任務，帶回素材 | 免費（需等待） |
| 隨從療養 | 回復在戰鬥中受重傷的隨從（HP 歸零後） | 20g |

## 八、龍人族長老 （Wyverian Elder）

**定位：** 龍人族長老 / 怪物知識與古代秘聞提供者

**姓名：** 蒼空爺爺（通稱）

**性格：** 古怪神秘，說話常常讓人摸不著頭腦，偶爾會說著說著就睡著。但每當怪物威脅真正降臨，他的眼神就會變得如古龍一般深邃。他見過的事比這個世界上任何人都多，包括上次黑龍降臨。

**背景：**
龍人族（Wyverian）是與怪物共存了數千年的古老種族，蒼空爺爺已年過三百。他在結雲村定居的原因至今無人知曉，只知道每當有重大的怪物事件，他總是早在事件發生前就有所預感。傳說他年輕時曾親手記錄過黑龍的行為模式，那份紀錄現在被他藏在某處。

**台詞：**
- 「（半睡半醒地）嗯……啊，是你啊……你來問怪物的事？好，坐，不過老夫先說，免費的消息不值錢，值錢的消息老夫不一定要說……嗯……（打盹）」
- 「那隻轟龍……（睜開眼，突然清醒）……不是普通的轟龍。牠的龍元素比同類強三倍。你要帶龍抗藥劑，帶很多。」
- 「黑龍……（長時間沉默）……孩子，有些怪物不是用來殺的。牠的存在，是世界平衡的一部分。記住這句話，有一天你會需要它。」
- 「你知道怪物為什麼如此強大嗎？因為牠們每一隻，都在拼命活著。而你，也是。這就是為什麼你們之間的狩獵，是如此……崇高。」

**服務內容：**

| 服務項目 | 說明 | 費用 |
|----------|------|------|
| 怪物圖鑑諮詢 | 查詢目標怪物弱點、抗性、行動模式（+3 於怪物知識檢定） | 15g |
| 古代知識 | 探索地圖的古代遺跡、秘密通道情報 | 25g |
| 古龍情報 | 古龍怪物的特殊能力、行動預測（僅限古龍等級任務） | 50g |
| 黑龍傳說 | 黑龍的神話記載與特殊應對方式（劇情觸發） | — |
| 素材鑑定（古代） | 鑑定古龍或黑龍級別的神秘素材 | 免費（他很感興趣） |
| 古代地圖判讀 | 解讀古代遺跡中的龍人族文字與地圖 | 20g |
