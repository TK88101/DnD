# 血源詛咒 — 敵人

> DM 根據玩家等級和區域選擇敵人。血源的敵人分為四大類：野獸、瘋狂市民、教會敵人、上位者。
> 記住：在亞南，每一個敵人都曾經是人。

```

祭壇前跪著一個身影——穿著白色長袍的女人。
她在祈禱。你能聽到她的聲音——破碎的、絕望的禱詞。

然後她停下了。

她的身體開始變化。骨骼在皮膚下扭曲生長。
長袍被撕裂。白色的毛髮從皮膚下刺出。
她轉過身來。

那已經不是一張人類的臉了。
```

```
它用整個存在看著你。

這就是一切的源頭。獵人夢境。血之醫療。瘟疫。
一切。

而現在，你要面對它。
```
