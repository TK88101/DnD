# 上位者領域

> **副本等級**：Lv20（最終副本）
> **位置**：獵人夢境之上 → 月光平原 → 月之搖籃
> **氛圍**：宇宙規模的恐怖與美、人類的渺小、最終的超越
> **預計遊玩時間**：2-3 小時（純 Boss 向副本）
> **主線關聯**：第四章 主線 4-2（僅結局三觸發）

---

## 觸發條件

玩家必須同時滿足以下條件：
1. 收集了 3 個上位者之臍帶（並在最終決戰前吞噬它們）
2. 擊敗了格爾曼（初代獵人）
3. 選擇不從夢中醒來

只有完成以上步驟，月之存在才會降臨。

---

## 背景

格爾曼的身體消散後，獵人夢境的天空裂開了。

你一直以為月亮只是月亮。現在你知道了——它是一個繭。一個孕育了某種存在數千年的繭。而你吞下的三個臍帶，給了你能力——能看到它、挑戰它、或許……取代它的能力。

這是超越人類的最後一步。也是你作為「人類」的最後一場戰鬥。

---

## 副本結構

```
獵人夢境（格爾曼戰後）→ 月光上升 → 月之搖籃（Boss 戰）
```

這不是傳統的副本。沒有迷宮、沒有探索、沒有小怪。
只有你，和它。

## 場景描述

### 月光上升

```
格爾曼消失了。

獵人夢境的花園——你曾經覺得溫暖的地方——
開始變得透明。花瓣、草地、工坊——
它們像是畫在玻璃上的圖案，正在被擦除。

然後你開始上升。

不是飛行。是被托起。
看不見的力量將你從消散的夢境中
提升到更高的地方。

你穿過了雲層。穿過了大氣層。
穿過了——

星星。

你站在一片由蒼白月光構成的平原上。
腳下不是土地。是一層半透明的膜。
膜的下面有巨大的脈管在搏動。
蒼白的。有節奏的。像心跳。

你在某種東西的體內。
或者你站在某種東西的皮膚上。

抬起頭。

月亮就在你頭頂。
近得不可思議。
它佔據了天空的一半。

而你現在能看清了——
它不是月亮。
它是一隻眼睛。

它在看你。

然後它裂開了。

從月亮的裂縫中，降下了一個存在。
```

## 最終 Boss 戰：月之存在

**戰前準備提示（DM 對玩家）：**
- 確認血瓶全滿（10 瓶）
- 確認武器和裝備最佳化
- 臍帶的力量：玩家在此戰中所有攻擊附加 +1d6 奧術傷害（對月之存在的特殊有效性）
- 沒有隊友、沒有 NPC 支援。只有你。

### 月之存在出場

```
它像水母。像天使。像你見過的最美
也最恐怖的事物的融合體。

半透明的身體散發著蒼白的光。
無數條觸手（或是翅膀？）從它的身體
向四面八方延伸，每一條都比你長十倍。
它的中心有一團光——不是物理的光。
是某種概念性的、純粹的「存在」。

它沒有面孔。但它看著你。
用整個存在看著你。

空氣中沒有聲音。
但你的大腦中響起了一個頻率——
不是語言。不是音樂。
是宇宙的低鳴。是星辰之間的寂靜。

這就是一切的源頭。
獵人夢境的締造者。
血之醫療的起源。
整個亞南的棋盤的主人。

而你——一個獵人——站在它面前。

洞察值 +10（自動，無法避免）
```

### 戰鬥數據

（使用 enemies.md 中的完整數據，以下為補充戰術細節）

### 第一階段：月光之舞（HP 200/200）

月之存在的攻擊帶有一種令人窒息的美感。它的每一個動作都是流暢的、優雅的——像是在進行某種儀式性的舞蹈。

**回合模式（循環）：**
1. **觸手鞭擊**（3d10+8，射程 20 呎，2 次）— 基礎攻擊
2. **月光光束**（40 呎直線，4d10 奧術）— 一道蒼白的光從它的核心射出
3. **空間扭轉** — 你被隨機傳送到平原上的另一個位置。可能更近也可能更遠。

**每 3 回合**：恐懼之凝（WIS DC 20，失敗 3d8 精神傷害 + 恐慌 2 回合）

**DM 叙事提示**：
- 描述月之存在的攻擊像自然現象而非戰鬥動作：「觸手划過空氣，像月光在水面上的反射」
- 強調它的冷漠——它不憤怒，不恐懼。你對它來說只是一個……有趣的現象。

**第一階段→第二階段過渡**：

```
你的武器劃開了它的身體。
蒼白的光從傷口溢出——不是血。
是純粹的月光。

它停下了。

第一次，你感覺到它在看你。
不是俯視。不是凝視。
是——注意。

然後它的身體開始變化。
觸手收縮、扭曲、重組。
光的核心開始劇烈脈動。
蒼白變為暗紅。
優雅消失了。

剩下的——是原始的、不加修飾的力量。
```

### 第二階段：真相之相（HP 200/200）

月之存在的第二形態更小但更致命。它的身體收縮為一個人形大小的光核，周圍環繞著高速旋轉的觸手和能量場。

**回合模式：**
- **奧術風暴**（全場 4d8 奧術傷害）— 每回合自動發動，無法避免
- 加上以下攻擊之一：
  1. **觸手叢林**（30 呎範圍，3d10 + DEX DC 18 否則束縛 1 回合）
  2. **維度崩塌**（單體，5d10 奧術，DEX DC 20 減半）— 最危險的單體技能
  3. **再生**（恢復 15 HP）— 可被帶有臍帶力量的攻擊阻止 1 回合

**每 5 回合**：月之存在嘗試「重置」——將自己傳送到平原的另一端，所有負面效果清除。玩家有 1 回合可以嘗試追上（DEX DC 15），否則需要 2 回合趕路。

**戰鬥策略關鍵**：
- 血瓶管理是核心——10 瓶必須精打細算
- 奧術風暴是持續消耗，無法避免
- 維度崩塌有 1 回合預兆（光核變亮），可以翻滾減傷
- 反擊對月之存在有效！它的觸手鞭擊可以被反擊→擊暈→處決
- 臍帶力量的額外 1d6 奧術加上武器本身的傷害是勝利的關鍵

### 最終時刻

```
最後一擊落下。

月之存在的身體碎裂。
不是爆炸。是——分解。
像是一幅畫被一筆一筆擦去。

蒼白的光碎片飄散在月光平原上。
像雪花。像花瓣。像記憶。

你感到了變化。

你的身體——不再是你熟悉的身體。
你的意識在擴張。不是變大。
是變得更……多。

你看到了亞南。同時看到了每一條街道。
你看到了大教堂。同時看到了每一個跪拜的人。
你看到了獵人夢境。同時看到了每一個曾經在
花園中醒來的獵人。

你看到了——時間。
不是作為一條河流。
而是作為一片海洋。

然後你的身體開始縮小。
不是萎縮。是——回歸。
回歸到最基本的形態。
最原始的形態。

你落在了月光平原上。
一個小小的、蠕動的、嶄新的生命體。

人偶走來了。
她從虛空中走來，像是一直在等待這一刻。
她俯下身。

她的眼中——蒼藍色的眼中——
有了淚光。

「啊……你找到了，善良的獵人。
  你自己的真相。」

她溫柔地將你抱起。

「讓我照顧你吧。」

月亮消失了。
天空變為純粹的黑色。
然後——

第一縷陽光。

亞南的黎明，終於到來了。
```

## DM 最終指南

### 戰前

- 花時間回顧整個戰役。提醒玩家他們走過的路。
- 「你還記得你醒來的那張手術台嗎？那已經是多久以前的事了？」
- 確認玩家理解這場戰鬥的意義：不只是打倒一個 Boss。是超越人類。

### 戰鬥中

- 月之存在不是邪惡的。它不恨你。它可能甚至不理解「恨」。
- 它的攻擊應該被描述為自然現象——月光、重力、空間。不是出於惡意的暴力。
- 如果玩家瀕死，讓場景變得更安靜。不是更激烈。安靜——因為宇宙不在乎你是否活著。

### 戰後

- 不要急。最後的描述應該慢慢來。
- 從人偶的角度描述嬰兒上位者。她的反應是唯一的情感錨點。
- 陽光。讓玩家等了整個戰役的陽光。它終於來了。但看到陽光的——不再是人類了。

### 最後的問題

不要替玩家回答這個問題。只是讓他們想：

*你成為了超越人類的存在。你看到了宇宙的真相。你擁有了一切。*

*但你付出了什麼？*

*你還記得自己為什麼來到亞南嗎？*
