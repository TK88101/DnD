# 亞南城區

> **副本等級**：Lv1-8
> **位置**：中央亞南 → 大橋 → 大教堂區
> **氛圍**：維多利亞哥特街道、篝火與嚎叫、瘋狂市民、初入地獄
> **預計遊玩時間**：3-4 小時
> **主線關聯**：第一章全部

---

## 背景

這是獵人踏入亞南的第一個夜晚。從約瑟夫卡診所走出門的那一刻起，你就身處狩獵之中。中央亞南的街道曾經是繁華的商業區——現在只有篝火、路障、和想要殺你的一切。

你的目標：活過今夜，到達大教堂。

---

## 區域地圖概述

```

                                                    ↓

```

## 區域詳情

### 1. 約瑟夫卡診所（起始區域）

```
你踏出手術室。走廊昏暗，只有盡頭一盞搖晃的油燈
提供微弱的光芒。牆壁上有抓痕——不是人的指甲
留下的。太深了。太寬了。

樓梯向下延伸。你聽到了一聲低吼。
```

- **教學區域**：1 隻獸化村民（HP 15，弱化版——讓玩家學習戰鬥）
- 如果玩家死亡，傳送到獵人夢境（教學復活機制）
- 搜索（INT DC 10）：手術室桌上有一瓶額外的血瓶

### 2. 診所小巷

```
狹窄的巷道夾在兩棟高聳的建築之間。
頭頂是被煙霧遮蔽的暗橙色天空。
地上散落著碎玻璃和被丟棄的家具。

巷道的盡頭，一群人圍著一堆篝火。
他們手持火把和叉子。他們看到了你。

「又一個外來者！殺了它！它是野獸的同類！」
```

- **遭遇**：武裝市民 x4（弱，主要是教學——學習群體戰鬥）
- 市民可以被嚇退（CHA DC 12）或繞過（DEX DC 13 潛行）
- 巷道中有 1 隻獸化犬在陰影中等待

### 3. 中央亞南廣場（燈火 1）

```
你踏入了一個開闊的廣場。
中央是一個巨大的篝火——
燃燒的不只是木頭。你看到了椅子、書桌、
還有你不想仔細看的東西。

廣場四周的門窗全部關閉。
但你能聽到門後的聲音——祈禱、哭泣、瘋笑。

一根石燈柱矗立在廣場角落。
它發出微弱的、溫暖的光芒。
不是火光。是別的什麼。
當你靠近它時，你感到一股平靜。

「點亮燈火吧，獵人。」——格爾曼的聲音在耳邊。
```

- **燈火**：點亮後成為復活點。全隊 HP 和血瓶補滿。但周圍敵人也會重生。
- **探索**：
  - 廣場周圍有 3 扇可以互動的門（都不會打開，但門後的 NPC 會說話）
    - 門 1：「離開這裡！不要靠近！你身上有血的味道！」
    - 門 2：「這是教會的報應……我們不該接受那些血……」
    - 門 3：（只有哭泣聲）
  - 搜索（INT DC 12）：廣場角落有一具屍體，身上有 30g 和一封信（提到大教堂的聖者會保護市民）

### 4. 下水道（可選區域）

```
鐵柵門嘎吱一聲打開。一股腐臭的氣味衝面而來。
石階向下延伸到黑暗中。你的手電筒——
不對，你沒有手電筒。你只有火把。

下水道的水是暗紅色的。
你不想知道為什麼。
```

- **環境**：部分區域水深及腰（移動速度減半）、昏暗（無光源則攻擊劣勢）
- **敵人**：獸化村民 x3、巨型老鼠 x5（HP 8，攻擊 1d4，群體）
- **小 Boss**：血之飢渴者（古老的獸化者，HP 80，AC 13）
  - 攻擊：利爪 2d6+3（2次）/ 血之吸取（命中後恢復等量 HP）/ 嚎叫（恐懼）
  - 弱點：火焰 +75%
- **獎勵**：血之回響 500、下水道寶箱（火焰紙 x3 + 血石碎片 x1）

### 5. 獵犬巷

```
巷道變寬了，但這不是好消息。
更寬意味著更多的東西可以藏在這裡。

你聽到了鏈條的叮噹聲。然後是犬吠。
不是普通的犬吠——是撕裂喉嚨的嘶吼。

一個高大的身影從陰影中走出。
他牽著三隻獸化犬。犬的口中流著黑色的涎液。

「放。」他說。

鏈條解開了。
```

- **遭遇**：獵犬管理員 x1 + 獸化犬 x3
- **策略提示**：先殺管理員可能讓犬陷入混亂（WIS DC 13 判定，成功則犬互相攻擊 1 回合）
- 巷道中有多個可利用的環境：油桶（攻擊引爆，10 呎 2d6 火焰）、木板牆（可推倒砸敵人 2d6）

### 6. 大橋（燈火 2）

```
你站在大橋的一端。
橋面寬闊，兩側是石雕欄杆——
天使和惡魔交替排列，在火光中投下扭曲的影子。

橋的另一端是大教堂區的入口。
但在你和入口之間，有一個東西。

它比任何你見過的獸化者都大。
像是一隻巨大的犬——不，是狼。
不，它太大了，不是任何已知的動物。
它佔據了整個橋面。
它的眼睛是暗紅色的。
它看到了你。
```

- **Boss 戰（可跳過）**：大橋野獸（HP 100，AC 13）
  - 攻擊：爪擊 2d8+4 / 衝撞（橋面全寬，3d8，DEX DC 15 閃避）/ 咆哮（恐懼）
  - 弱點：火焰 +50%、橋欄杆可阻擋衝撞（掩體 +4）
  - **繞路選項**：橋下有一條隱藏的維修通道（DEX DC 14 發現並通過）
- 燈火在橋的中央（戰鬥前可點亮）

### 7. 大教堂區（探索區域）

- 教會巡邏兵 x多、教會巨人 x1
- 多個可探索的側翼區域
- 教會商店（需教會徽章，擊敗巡邏隊長後獲得）

### 8. 大教堂（Boss 區域 + 燈火 3）

**Boss 戰：牧師阿梅利亞**（詳見 enemies.md）

## DM 氛圍指南

- **漸進式恐怖**：從診所的安靜到街道的混亂到下水道的黑暗到大橋的壯觀。逐步升級。
- **音景**：永遠有背景音——犬吠、教堂鐘聲、篝火噼啪、遠處的尖叫、門後的低語。
- **戰鬥節奏**：前半段頻繁但簡短的小戰鬥，培養玩家的反應。後半段減少頻率但增加難度。
- **NPC 互動**：門後的聲音非常重要。它們是這個世界仍有人類存在的證據。讓玩家感到——他們在為這些聲音而戰。
- **Boss 戰**：阿梅利亞的戰鬥應該讓玩家感到悲傷。她在變身前的禱告聲應該讓人心碎。她不是怪物。她是一個失敗的聖者。
