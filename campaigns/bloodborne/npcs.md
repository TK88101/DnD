# 血源詛咒 — NPC

> DM 使用此文件管理非玩家角色。在亞南，大部分 NPC 都躲在緊閉的門後說話。出來的人……通常都有問題。

---

## 1. 人偶

*「歡迎回家，善良的獵人。讓我為你分擔些什麼吧。」*

### 基本資料

| 項目 | 內容 |
|------|------|
| 身份 | 獵人夢境的守護者 / 血之回響的管理者 |
| 外貌 | 真人大小的瓷白色人偶，維多利亞式服裝，銀色長髮，蒼白而美麗的面容。她的關節處可以看到人偶的接縫。她的眼睛是蒼藍色的——有時你覺得那是玻璃，有時你覺得那是活的。 |
| 性格 | 溫柔、順從、神秘。她的存在帶來安慰，但你永遠不知道她到底是什麼。 |
| 位置 | 獵人夢境（永遠在那裡） |

### 故事角色

人偶是獵人在這漫長夜晚中唯一恆定的存在。她管理獵人夢境中的升級系統——消耗血之回響提升獵人的等級。她對每個獵人都一樣溫柔。她不判斷、不質疑。她只是在那裡，等你回來。

### 對話風格
```
*她微微鞠躬，動作像是被精心設計的——但又不完全是*

「善良的獵人，你受傷了嗎？
  請把你的血之回響交給我。
  我會……好好利用它們。讓你變得更強。」

*她的手指觸碰你的額頭。冰涼的。但不是不舒服的。*

「願你在這漫長的夜晚中找到價值。」
```

### DM 資訊（玩家不可見）

- 人偶是以拜爾根沃斯學者瑪麗亞的形象製造的——格爾曼對瑪麗亞有深深的執念
- 她擁有獨立的意識，但不確定自己是否「活著」
- 如果玩家洞察值超過 60，她會開始說出更多關於獵人夢境本質的話
- 她可能是理解月之存在的關鍵——但她自己不知道

### 互動能力

| 服務 | 效果 |
|------|------|
| 升級 | 消耗血之回響提升等級 |
| 血瓶補充 | 離開夢境時自動補滿 |
| 對話 | 提供溫暖的鼓勵和偶爾的神秘暗示 |
| 冷血寶石鑲嵌 | 協助將冷血寶石鑲嵌至武器（武器強化） |

## 2. 格爾曼——初代獵人

*「今夜，格爾曼加入狩獵。」*

### 基本資料

| 項目 | 內容 |
|------|------|
| 身份 | 獵人夢境的主人 / 第一個獵人 / 月之存在的囚徒 |
| 年齡 | 外表約 70 歲，實際年齡不可計算（他被困在夢中已數個世紀） |
| 外貌 | 乾瘦的老人，坐在輪椅上。灰白的長髮遮住半張臉。穿著老舊的獵人裝束。他的右腿是義肢。手邊永遠放著一把未展開的葬送刃。 |
| 性格 | 疲憊、慈祥、沉重。他的話語中有數個世紀的孤獨和後悔。 |
| 位置 | 獵人夢境 · 工坊二樓（大部分時間在睡覺） |

### 故事角色

格爾曼是整個故事的隱藏核心。他創建了最初的獵人工坊，訓練了第一代獵人。但在某一夜，他與月之存在達成了契約——他的靈魂被困在獵人夢境中，永遠指導新的獵人。他想要自由。

### 對話風格
```
*他抬起疲憊的眼睛看你。那雙眼睛看過太多的死亡。*

「又一個新的獵人嗎……嗯……去吧。
  殺幾隻野獸。血之意志會引導你。」

*他的手摸了摸身邊的武器*

「當這一切結束的時候……
  如果你願意的話，回來找我。
  我有一個……禮物。給你。」

*他閉上眼睛，又沉入了不安的睡眠*
```

### DM 資訊（玩家不可見）

- 格爾曼的「禮物」是死亡——他會在最終章提出斬首玩家，讓他們從夢中醒來（結局一）
- 他的葬送刃是遊戲中最強的武器之一
- 他知道月之存在的存在，但無力對抗它
- 如果玩家拒絕他的提議，他會變成最終 Boss 之一

### 戰鬥數據（最終章 Boss）

| 屬性 | 值 |
|------|---|
| HP | 200 |
| 閃避值 | 20（獵人直覺 +6） |
| 攻擊 | 葬送刃·鐮刀形態 3d10+8（2 次/回合）/ 葬送刃·直劍形態 2d8+6（3 次/回合）/ 風之衝擊（30 呎直線 4d8，DEX DC 18 減半） |
| 特殊 | 快速步伐（每回合免費翻滾 2 次）、獵人直覺（無法被突襲）、不朽意志（HP<25% 時攻擊 +5）、老獵人的驕傲（免疫恐懼和控制） |

## 3. 艾琳——鴉羽獵人

*「今夜有獵人犯了不可饒恕的罪。我來清理他們。」*

### 基本資料

| 項目 | 內容 |
|------|------|
| 身份 | 獵人殺手 / 教會的清道夫 |
| 年齡 | 約 35 歲 |
| 外貌 | 穿著黑色鴉羽外套，面具遮住下半張臉。只露出一雙冷靜的深色眼睛。身形纖細但動作像獵豹。 |
| 性格 | 冷靜、務實、有自己的正義感。她殺人不眨眼，但只殺「應該被殺的人」。 |
| 位置 | 各區域隨機出現（她在追蹤墮落的獵人） |

### 故事角色

艾琳是教會的秘密清道夫——專門獵殺那些獸化或失控的獵人。她是玩家在亞南城區中少數友善的面孔之一。她會在關鍵時刻出現提供資訊和幫助。

### 對話風格
```
*她從屋頂的陰影中跳下，落地幾乎無聲*

「你是新來的獵人？嗯，還活著，不錯。」

*她擦拭匕首上的血*

「聽好了。不是所有獵人都是好人。
  有些人獵殺的時間太長了。他們開始享受殺戮。
  然後他們就變成了野獸——比街上那些更危險的野獸。

  如果你遇到了不對勁的獵人，不要猶豫。
  先下手為強。」
```

### DM 資訊（玩家不可見）

- 她的任務線是追蹤一名叫「血之烏鴉」的瘋狂獵人——最終可能需要玩家的幫助
- 如果她的任務線完成且她存活，她會在最終章成為盟友
- 如果任務線失敗（或玩家攻擊她），她會獸化並成為敵人

### 互動能力

| 服務 | 效果 | 條件 |
|------|------|------|
| 情報 | 提供當前區域的獵人和敵人資訊 | 遭遇時自動 |
| 協助戰鬥 | 在特定 Boss 戰中作為盟友參戰 | 完成她的支線任務 |
| 鴉羽技巧 | 教導一個特殊技能：「鴉羽閃避」（翻滾距離 x2，1 次/場） | 好感度高 |

## 4. 阿爾弗雷德——教會執行者

*「教會的意志是絕對的。凱恩赫斯特的異端必須被淨化。」*

### 基本資料

| 項目 | 內容 |
|------|------|
| 身份 | 教會的忠實執行者 / 處刑人 |
| 年齡 | 約 30 歲 |
| 外貌 | 金色短髮、蒼白的面容、穿著教會處刑者的金色長袍。肩上扛著一把比他還高的教會輪鉅。他的笑容親切，但眼中有狂熱的光。 |
| 性格 | 虔誠、友善（對同道中人）、對異端絕不留情。他的信仰是真摯的——這正是他危險的地方。 |
| 位置 | 大教堂區 / 凱恩赫斯特城（任務線後期） |

### 故事角色

阿爾弗雷德是教會在獵人中的代表。他為玩家提供關於教會和凱恩赫斯特的背景資訊。他的任務線涉及凱恩赫斯特城堡和安娜莉絲女王。

### 對話風格
```
*他露出燦爛的笑容，陽光般的——在這血腥的夜晚中格格不入*

「啊！一位新獵人！太好了！
  教會的工作永遠需要更多的幫手。」

*他的表情突然嚴肅*

「但你要小心。這座城市的腐敗不僅僅是野獸。
  凱恩赫斯特的叛徒們……他們用污穢之血褻瀆了
  教會的教導。他們必須被消滅。
  每一個。」

*微笑重新回到他臉上。但那微笑比之前更冷了。*
```

### DM 資訊（玩家不可見）

- 他的最終目標是殺死凱恩赫斯特女王安娜莉絲
- 如果玩家幫助他到達凱恩赫斯特，他會執行殺戮——這可能會阻斷獲取「上位者之臍帶」的途徑
- 玩家需要在他和安娜莉絲之間做出選擇

## 5. 安娜莉絲——凱恩赫斯特女王

*「啊……你願意成為我的騎士嗎？用你的血，為我而戰？」*

### 基本資料

| 項目 | 內容 |
|------|------|
| 身份 | 凱恩赫斯特城堡的女王 / 污穢之血的守護者 |
| 年齡 | 不明（數百年，血維持了她的生命） |
| 外貌 | 蒼白如雪的肌膚、銀白色長髮、穿著華麗但陳舊的宮廷禮服。她的美麗帶有一種不自然的永恆感。她的王座上覆蓋著乾涸的血。 |
| 性格 | 優雅、冷漠、渴望——她渴望一個可以為她而死的騎士 |
| 位置 | 凱恩赫斯特城堡（需通過特殊途徑到達） |

### 故事角色

安娜莉絲是凱恩赫斯特貴族的最後一人。她擁有一個上位者嬰兒的臍帶——第三個結局所需的三個臍帶之一。她可以讓玩家加入「污穢之血」陣營。

### 對話風格
```
*她從王座上微微俯身，蒼白的手指伸向你*

「來，讓我看看你。」

*她的眼睛——紅色的虹膜——掃過你的全身*

「你的血……嗯。有趣。
  不像其他獵人。你的血裡有……潛力。」

*她收回手，端莊地坐正*

「我需要一個騎士。一個願意為我收集
  血之意志的騎士。作為回報——
  我的知識，我的血，都是你的。
  這筆交易……你滿意嗎？」
```

### DM 資訊（玩家不可見）

- 加入她的陣營可獲得「污穢之血」能力（攻擊命中後有概率吸血）
- 她的臍帶碎片是第三結局的關鍵物品
- 如果阿爾弗雷德到達這裡且玩家不阻止，她會被殺（但可以復活——如果玩家有特殊物品）

## 6. 帕奇——蜘蛛

*「嘿嘿嘿……又來了一隻蒼蠅。」*

### 基本資料

| 項目 | 內容 |
|------|------|
| 身份 | 亞南的生存者 / 商人 / 騙子 |
| 年齡 | 不明 |
| 外貌 | 禿頭、瘦削、不自然的笑容。穿著簡陋但乾淨的衣物。他的動作像蜘蛛一樣——輕巧、精準、讓人不舒服。洞察值 40 以上時，可以看到他身後有蜘蛛腿的幻影。 |
| 性格 | 狡猾、自私、膽小——但有一種奇怪的可靠性。他騙你，但他也賣你需要的東西。 |
| 位置 | 禁忌森林 / 噩夢邊境（他總是在最不該出現的地方出現） |

### 故事角色

帕奇是一個反覆出現的角色——有時是敵人（把你推下懸崖），有時是商人（賣你稀有物品）。他是亞南混亂中的一個常數——不好不壞，只是……帕奇。

### 對話風格
```
*他從陰影中走出來，雙手張開，一臉無辜*

「哎呀呀，我們又見面了！純粹的巧合！
  我發誓上次那個懸崖的事是個意外！」

*他注意到你手中的武器*

「好好好，別那麼緊張嘛。
  我這裡有好東西——你需要的好東西。
  便宜賣你。因為我們是……朋友嘛。」

*他的笑容在燭光中看起來像骷髏*
```

### DM 資訊（玩家不可見）

- 他實際上是上位者的眷族（蜘蛛型），但他的人性大部分還在
- 他的陷阱總是在可存活的範圍內——他不真的想殺你，只是覺得「很好笑」
- 在最終章，如果你之前沒有攻擊他，他會以商人身份出現在最危險的地方

### 互動能力

| 服務 | 效果 | 備註 |
|------|------|------|
| 商店 | 出售稀有物品和彈藥（加價 50%） | 他是騙子，但貨真價實 |
| 情報 | 偶爾提供有用的（但不完整的）情報 | 50% 概率是真的 |
| 陷阱 | 第一次遇到時有 50% 概率把你推下去 | 可以 WIS DC 14 預判 |
