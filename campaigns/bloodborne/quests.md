# 血源詛咒 — 任務

> DM 使用此文件管理主線和支線任務。血源的任務圍繞「一夜的狩獵」展開。
> 沒有人委託你做這些事。你這麼做是因為——如果你不做，天亮就不會來。

---

## 主線任務

### 第一章：狩獵之夜（Lv1-8）

---

#### 主線 1-1：醒來

| 項目 | 內容 |
|------|------|
| 觸發 | 戰役開始 |
| 目標 | 在約瑟夫卡診所醒來，獲取武器，走出去面對亞南 |
| 地點 | 約瑟夫卡診所 → 中央亞南 |
| 建議等級 | 1 |

**任務流程：**
1. 獵人在手術台上醒來。地上有血跡和一張紙條（格爾曼的訊息）。
2. 診所一樓有一隻獸化村民（教學戰鬥——玩家可能第一次死亡）。
3. 死亡後在獵人夢境醒來。遇見人偶和格爾曼。格爾曼解釋基本規則。
4. 從夢境返回，正式踏入中央亞南的街道。

**獎勵：**
- 起始武器和手槍
- 了解獵人夢境和燈火系統
- 血之回響 50

---

#### 主線 1-2：中央亞南的狩獵

| 項目 | 內容 |
|------|------|
| 觸發 | 完成主線 1-1 |
| 目標 | 穿越中央亞南，到達大橋 |
| 地點 | 中央亞南各區域 |
| 建議等級 | 1-4 |

**任務流程：**
1. 探索中央亞南的街道。遭遇武裝市民、獸化犬、獸化村民。
2. 發現第一個燈火（中央亞南廣場）。
3. 在下水道入口遭遇大型獸化者（小型 Boss 戰）。
4. 到達大橋前，遭遇牧羊人（友善 NPC，提供關於大教堂的情報）。
5. 大橋上有一隻巨型獸化犬守衛——可以戰鬥或找到繞路方式。

**敵人：** 武裝市民 x多、獸化犬 x多、獸化村民 x多、大型獸化者 x1

**獎勵：**
- 血之回響 500
- 燈火：中央亞南廣場
- 通往大教堂區的路線

---

#### 主線 1-3：大教堂的聖者

| 項目 | 內容 |
|------|------|
| 觸發 | 到達大教堂區 |
| 目標 | 進入大教堂，面對牧師阿梅利亞 |
| 地點 | 大教堂區 → 大教堂 |
| 建議等級 | 5-8 |

**任務流程：**
1. 探索大教堂區。遭遇教會巡邏兵和教會巨人。
2. 發現教會的不正常之處——巡邏兵在殺市民，不只是野獸。
3. 在大教堂前遇到阿爾弗雷德（友善 NPC，提供教會資訊）。
4. **Boss 戰：牧師阿梅利亞**（詳見 enemies.md）
5. 擊敗後，在祭壇後方發現上位者之臍帶 #1 和一段影像——教會的高層知道血之醫療的真相。

**獎勵：**
- 血之回響 3000
- 上位者之臍帶 #1
- 燈火：大教堂
- 解鎖：第二章路線（禁忌森林的路被打開）

### 第二章：禁忌的真相（Lv8-15）

---

#### 主線 2-1：深入禁忌森林

| 項目 | 內容 |
|------|------|
| 觸發 | 完成第一章 |
| 目標 | 穿越禁忌森林，找到通往拜爾根沃斯的路 |
| 地點 | 禁忌森林 |
| 建議等級 | 8-12 |

**任務流程：**
1. 禁忌森林是一座迷宮。WIS DC 14 導航，失敗迷路（額外遭遇）。
2. 遭遇森林中的獸化村民（比城市中的更危險）、蛇群、陷阱。
3. 在森林深處遇到隱藏村莊——村民全部獸化，但村莊教堂中有古籍（閱讀洞察 +5）。
4. **Boss 戰：暗影之獸 x3**（三隻一組，各有不同攻擊方式）
5. 擊敗後發現通往月側湖的秘密通道。

**敵人：** 蛇群 x多、獸化村民（強化）x多、暗影之獸 x3

**獎勵：**
- 血之回響 2000
- 古籍情報（洞察 +5）
- 通往拜爾根沃斯的路

---

#### 主線 2-2：月側湖之秘

| 項目 | 內容 |
|------|------|
| 觸發 | 完成主線 2-1 |
| 目標 | 在拜爾根沃斯發現血之醫療的起源，面對白痴蜘蛛羅姆 |
| 地點 | 拜爾根沃斯學院 → 月側湖 |
| 建議等級 | 12-14 |

**任務流程：**
1. 探索拜爾根沃斯學院。閱讀學者的研究筆記——發現上位者的存在。
2. 找到威廉大師（NPC，古老的學者，坐在輪椅上指向月側湖）。
3. 到達月側湖。湖面漆黑如墨。
4. **Boss 戰：白痴蜘蛛羅姆**（詳見 enemies.md）
5. **劇情轉折**：擊殺羅姆後，血月降臨。天空變為血紅色。世界永久改變。

**獎勵：**
- 血之回響 5000
- 羅姆的符文
- **世界狀態改變**：血月降臨，所有區域敵人 +10%，隱藏敵人出現
- 解鎖：第三章

### 第四章：月之存在（Lv18-20）

---

#### 主線 4-1：最終的夢境

| 項目 | 內容 |
|------|------|
| 觸發 | 完成第三章所有主線 |
| 目標 | 回到獵人夢境，面對格爾曼的提議 |
| 地點 | 獵人夢境 |
| 建議等級 | 18-20 |

**任務流程：**
1. 獵人夢境的花園正在燃燒。白色花瓣在火焰中飛舞。
2. 格爾曼站在花園中央，不再坐在輪椅上。他手持葬送刃。

```
「善良的獵人……你做得夠多了。
  夜晚即將結束。
  我可以讓你從這場夢中醒來。
  你會忘記一切——亞南、野獸、血。
  你會回到正常的世界。
  活下去。

  你只需要……低下頭。」
```

3. 玩家的選擇：
   - **選擇一：接受**（結局一）→ 格爾曼斬首獵人。獵人在亞南城外醒來，忘記一切。遊戲結束。
   - **選擇二：拒絕** → Boss 戰：格爾曼（詳見 NPC 戰鬥數據）
   - **選擇三：拒絕 + 已收集 3 個上位者之臍帶** → 先與格爾曼戰鬥，然後面對月之存在

**獎勵依結局而異**（見下方結局部分）

## 結局

### 結局一：亞南的清晨（順從）

```
刀鋒落下。乾淨俐落。

你感覺不到疼痛。只有一瞬間的寒冷。
然後——溫暖。

你醒了。
陽光照在你臉上。鳥兒在叫。
你坐在亞南城外的草地上。

你不記得為什麼來到這裡。
你不記得夜晚。不記得野獸。不記得血。

你只知道——你的病好了。
你活著。

陽光真好。
```

### 結局二：獵人夢境的守護者（反抗）

```
格爾曼倒下了。他的臉上有……釋然。

「終於……自由了……」

他的身體消散在月光中。
而你感到一股力量拉扯著你。

你坐下了。坐在格爾曼的輪椅上。
花園停止了燃燒。白色花瓣重新落下。

人偶走過來。
她微微鞠躬。

「善良的獵人。歡迎回家。」

你永遠無法離開獵人夢境了。
但至少——你可以為下一個獵人指路。

下一個漫長的夜晚即將開始。
```

### 結局三：上位者的誕生（超越）

```
月之存在的身體碎裂。
月光構成的碎片像雪花一樣飄落。

你感到自己的身體在變化。
不是痛苦。是……擴張。

你的意識不再局限於一具軀殼。
你看到了——一切。
亞南。世界。宇宙。時間的河流。

人偶走過來。
她俯身看著地面上一個小小的、蠕動的東西。

一個嬰兒？不。一個上位者的幼體。
它曾經是你。或者說，你曾經是它。

人偶溫柔地將它抱起。

「啊……你找到了，善良的獵人。
  你自己的真相。」

她的眼中——那雙蒼藍色的眼中——
第一次有了淚光。

「讓我照顧你吧。」
```
