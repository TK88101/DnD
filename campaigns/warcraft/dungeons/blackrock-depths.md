## 背景故事

黑石深淵是暗鐵矮人一族在黑石山深處建造的龐大地下城市。數百年前，暗鐵矮人的巫師薩弗隆召喚了火焰之王拉格納羅斯，結果不僅沒能控制這位元素領主，反而讓整個暗鐵一族淪為火焰的奴僕。

如今，弗萊拉斯大帝統治著這座被熔岩環繞的都市，表面上是暗鐵矮人的皇帝，實際上不過是拉格納羅斯的傀儡。黑石深淵內部設有監獄、酒吧、競技場、以及皇帝的宮殿，每一處都隱藏著暗鐵矮人的陰謀與秘密。

冒險者們接到消息——數名聯盟或部落的偵察兵在深淵中失蹤，必須深入這座地底城市查明真相，並阻止弗萊拉斯大帝為拉格納羅斯所進行的邪惡儀式。

## 地圖結構

黑石深淵擁有三條主要路線，冒險者可以自由選擇：

```
                    [入口大廳]
                    /    |    \
                   /     |     \
          [監獄路線]  [主路線]  [酒吧路線]
              |        |          |
         [R1:監獄]  [R4:通道]  [R6:黑鐵酒吧]
              |        |          |
         [R2:審訊室] [R5:競技場] [R7:釀酒廠]
              |        |          |
              +--------+----------+
                       |
                  [R8:帝王大廳]
```

### 路線說明
- **監獄路線**：難度較低，可以救出NPC獲得額外獎勵
- **酒吧路線**：有趣的角色扮演機會，可能避免部分戰鬥
- **主路線**：直面競技場挑戰，獲得最多戰鬥經驗

## 房間描述

### 房間1：暗鐵監獄（監獄路線）

> 鐵欄杆在熔岩的光芒下泛著暗紅色的光澤。數十個牢房排列在走廊兩側，其中傳來微弱的呻吟聲。空氣中瀰漫著腐爛和硫磺的氣息。

**環境效果**：昏暗光線（disadvantage on Perception checks unless darkvision）

**敵人：暗鐵獄卒**
| 屬性 | 數值 |
|------|------|
| HP | 45 |
| AC | 15 |
| 攻擊 | 鐵鞭 +7命中，2d6+3 鈍擊傷害 |
| 特殊 | 束縛之鏈：以附贈動作對一名目標施加束縛（DC 14 力量豁免解除） |

### 房間6：黑鐵酒吧（Boss 4 / 酒吧路線）

> 厚重的木門推開後，一股濃烈的麥酒香氣撲面而來。這是一間巨大的地下酒館，昏黃的燈光下，數十名暗鐵矮人正在豪飲。吧台後方，一名壯碩的矮人老闆正擦著酒杯，用充滿警惕的目光打量著你們。
