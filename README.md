# 🎲 D&D 無盡冒險 — 多人文字 RPG

基於瀏覽器的多人龍與地下城文字 RPG 遊戲。使用 Google Gemini AI 作為地下城主（DM），支持 1-8 人同時遊玩。

## 🎮 四大戰役

| 戰役 | 主題 | 特色 |
|------|------|------|
| ⚔️ 艾澤拉斯征途 | 魔獸世界 | 聯盟/部落陣營、12種族10職業、副本團本 |
| 🐙 迷霧深淵 | 克蘇魯神話 | 1920年代調查、理智值系統 |
| 🩸 血月獵殺 | 血源詛咒 | 哥特恐怖、變形武器 |
| 🐉 狩獵時刻 | 怪物獵人 | 狩獵→素材→鍛造、貓車機制、10種武器 |

## ✨ 核心功能

- **多人實時對戰** — WebSocket 連接，1-8 人同房遊玩
- **AI 地下城主** — Gemini 2.5 Flash 驅動的沉浸式敘事
- **角色創建系統** — 種族/職業/屬性分配（MH 簡化為選武器即玩）
- **D&D 5e 戰鬥** — 完整的掷骰判定、攻擊/技能/暴擊系統
- **MP 魔力系統** — 法術型職業的資源管理
- **動態難度** — 代碼根據隊伍人數自動縮放怪物數值
- **外掛記憶體** — 遊戲狀態持久化，對話歷史自動裁剪，token 消耗恆定
- **存檔/讀檔** — 隨時保存進度，跨房間讀取繼續
- **暫離系統** — 60 秒無操作自動 AFK，NPC 接管角色
- **缺席玩家處理** — 讀檔時自動偵測缺席玩家，角色由 NPC 控制
- **BGM 背景音樂** — YouTube 自動切換場景音樂（帶 fallback）
- **老闆鍵** — 按 Esc 切換偽裝畫面
- **角色顏色系統** — 每個角色獨立顏色，旁白白色，選項青色

### 怪物獵人專屬
- 🐱 **貓車機制** — 力盡3次任務失敗，怪物不回血
- ⚔️ **部位破壞** — 破壞頭/尾/翼影響怪物能力
- 🎯 **捕獲系統** — 跛行→陷阱→麻醉球，捕獲獎勵更豐富
- 🛒 **代碼控制商店** — 雜貨店由服務器代碼處理，不經過 AI
- 📈 **代碼控制升級** — EXP 達標自動升級，代碼擲骰算 HP

## 🚀 快速開始

### 環境需求
- Node.js 18+
- Google Gemini API Key

### 安裝

```bash
git clone https://github.com/你的用戶名/DnD.git
cd DnD/server
npm install
```

### 啟動

```bash
export GEMINI_API_KEY="你的API密鑰"
node relay.js
```

瀏覽器打開 `http://localhost:8080`

### 遠程遊玩（Cloudflare Tunnel）

```bash
cloudflared tunnel --url http://localhost:8080
```

分享生成的 URL 給朋友即可加入。

## 🎮 遊戲指令

| 指令 | 說明 |
|------|------|
| `開始遊戲` | 開始新遊戲 |
| `讀取 名字` / `讀檔 名字` | 讀取存檔 |
| `保存` / `存檔` | 保存遊戲 |
| `結束遊戲` / `退出` | 保存並結束 |
| `雜貨店` / `/shop` | 打開商店（代碼控制） |
| `/back` / `回來了` | 解除暫離狀態 |
| `Esc` | 老闆鍵（切換偽裝畫面） |

## 🏗️ 技術特點

- **Gemini 不可信原則** — 升級、商店、難度、選項編號全由代碼控制
- **選項轉換** — 玩家輸入數字，代碼查表轉成文字指令發給 Gemini
- **外掛記憶體** — 從 AI 回覆解析狀態（HP/金幣/物品），注入下次訊息
- **動態難度** — 代碼根據人數計算 HP 倍率和攻擊修正
- **心跳保活** — 15 秒 ping 防 Cloudflare 90 秒超時斷線
- **斷線重連** — 5 分鐘房間保留，主機/玩家可重連
