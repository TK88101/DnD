# 核心規則

> 所有戰役共用的基礎規則系統。各戰役的專屬規則見對應的規則文件。

---

## 一、屬性系統

### 六大屬性

| 屬性 | 縮寫 | 影響 |
|------|------|------|
| 力量 | STR | 近戰攻擊與傷害、負重、力量檢定（破門、攀爬） |
| 敏捷 | DEX | 遠程攻擊、AC（閃避）、先攻、敏捷檢定（潛行、開鎖） |
| 體質 | CON | 生命值、耐力、體質豁免（抗毒、抗疾病） |
| 智力 | INT | 法術攻擊（法師系）、知識檢定、調查 |
| 感知 | WIS | 察覺、洞察、治療、意志豁免 |
| 魅力 | CHA | 社交（說服、欺騙、威嚇）、部分魔法（術士、聖騎士） |

### 屬性調整值

```
調整值 = (屬性值 - 10) ÷ 2（向下取整）
```

| 屬性值 | 調整值 |
|--------|--------|
| 1      | -5     |
| 2-3    | -4     |
| 4-5    | -3     |
| 6-7    | -2     |
| 8-9    | -1     |
| 10-11  | 0      |
| 12-13  | +1     |
| 14-15  | +2     |
| 16-17  | +3     |
| 18-19  | +4     |
| 20     | +5     |

### 屬性範圍

- 最低：1（任何屬性降至 0 即角色無法行動）
- 最高：20（初始上限 15，通過裝備和增益可突破）

## 二、角色創建

### 點數購買法（27 點）

初始所有屬性為 8，花費點數提升：

| 目標屬性值 | 花費點數 |
|-----------|---------|
| 8         | 0       |
| 9         | 1       |
| 10        | 2       |
| 11        | 3       |
| 12        | 4       |
| 13        | 5       |
| 14        | 7       |
| 15        | 9       |

> 點數分配完成後，再加上種族屬性加成。

### 創建步驟

1. 選擇戰役
2. 選擇種族（獲得屬性加成和種族特長）
3. 選擇職業（決定 HP 骰、技能、裝備類型）
4. 分配 27 點屬性
5. 命名角色
6. 簡述背景（1-2 句，DM 可據此生成個人劇情）
7. 獲得職業起始裝備
8. 計算初始 HP = 職業 HP 骰最大值 + CON 調整值
9. 計算初始 AC = 10 + DEX 調整值 + 護甲加值 + 盾牌加值

## 三、戰鬥系統

### 戰鬥流程

```
1. 觸發戰鬥（遭遇敵人或主動進攻）
2. 掷先攻（所有參戰者）
3. 按先攻順序輪流行動
4. 每人每回合：1 個主要行動 + 1 個移動 + 1 個附加行動（如有）
5. 重複直到一方全滅/逃跑/投降
```

### 先攻

```
先攻值 = d20 + DEX 調整值
```

- 相同先攻值：玩家優先於敵人，NPC 隊友按職業定位排序（坦克 > DPS > 治療）
- 先攻決定後整場戰鬥不變

### 攻擊

```
攻擊檢定 = d20 + 攻擊調整值
```

- **近戰攻擊調整值** = STR 調整值 + 熟練加值
- **遠程攻擊調整值** = DEX 調整值 + 熟練加值
- **法術攻擊調整值** = INT 或 WIS 或 CHA 調整值 + 熟練加值（視職業）

命中條件：攻擊檢定 ≥ 目標 AC

### 熟練加值

| 等級 | 熟練加值 |
|------|---------|
| 1-4  | +2      |
| 5-8  | +3      |
| 9-12 | +4      |
| 13-16| +5      |
| 17-20| +6      |

### 傷害

```
傷害 = 武器傷害骰 + 屬性調整值
```

- 近戰武器：+ STR 調整值
- 遠程武器：+ DEX 調整值
- 法術：按法術說明

### 暴擊與大失敗

- **自然 20（暴擊）**：自動命中，傷害骰**投兩次**取總和（加值不翻倍）
- **自然 1（大失敗）**：自動未命中，DM 可描述一個小意外（武器脫手、絆倒等，不造成實際懲罰）

### AC（護甲等級）

```
AC = 10 + DEX 調整值 + 護甲加值 + 盾牌加值
```

- 重型護甲（鎖甲、板甲）：不加 DEX 調整值，但護甲加值更高
- 中型護甲（皮甲）：DEX 調整值上限 +2
- 輕型護甲（布甲）：完整 DEX 調整值

### 主要行動選項

每回合可選擇一個主要行動：

| 行動 | 說明 |
|------|------|
| 攻擊 | 用武器或徒手攻擊一個目標 |
| 施放技能 | 使用職業技能或法術 |
| 使用物品 | 使用藥水或其他消耗品 |
| 閃避 | 本回合所有針對你的攻擊檢定劣勢（掷兩次取低） |
| 援助 | 幫助一名隊友，給予其下次檢定優勢（掷兩次取高） |
| 脫離 | 安全撤離戰鬥（不觸發機會攻擊） |

### 機會攻擊

當敵人在你的近戰範圍內離開時，你可以進行一次免費攻擊（每回合限一次）。

## 四、技能檢定

### 通用公式

```
檢定結果 = d20 + 相關屬性調整值 + 熟練加值（如果擅長）
```

### 難度等級（DC）

| DC | 難度描述 | 範例 |
|----|---------|------|
| 5  | 極簡單 | 爬一面有梯子的牆 |
| 10 | 簡單   | 撬開一把普通鎖 |
| 12 | 普通   | 注意到隱藏在暗處的敵人 |
| 15 | 困難   | 說服一個警戒的衛兵讓你通過 |
| 18 | 很難   | 翻越一面光滑的高牆 |
| 20 | 極難   | 破解遠古符文密碼 |
| 25 | 近乎不可能 | 說服敵對首領投降 |

### 常用技能與對應屬性

| 技能 | 屬性 | 用途 |
|------|------|------|
| 運動 | STR | 攀爬、游泳、跳躍、推擠 |
| 特技 | DEX | 翻滾、平衡、逃脫束縛 |
| 隱匿 | DEX | 潛行、藏匿 |
| 開鎖 | DEX | 撬鎖、解除陷阱 |
| 調查 | INT | 搜索線索、研究文獻 |
| 奧秘 | INT | 辨識魔法、理解法陣 |
| 察覺 | WIS | 發現隱藏事物、感知危險 |
| 洞察 | WIS | 判斷 NPC 是否說謊 |
| 醫療 | WIS | 穩定瀕死隊友、辨識疾病 |
| 說服 | CHA | 友善地影響他人 |
| 欺騙 | CHA | 說謊、偽裝 |
| 威嚇 | CHA | 恐嚇、施壓 |

### 優勢與劣勢

- **優勢**：掷 2d20 取**高**值（有利條件時）
- **劣勢**：掷 2d20 取**低**值（不利條件時）
- 同時有優勢和劣勢 → 互相抵消，正常掷骰

## 五、生命值與死亡

### HP 計算

```
1 級 HP = 職業 HP 骰最大值 + CON 調整值
升級 HP = 擲職業 HP 骰 + CON 調整值（最低 +1）
```

### 職業 HP 骰

| HP 骰 | 職業類型 |
|--------|---------|
| d6     | 施法者（法師、術士） |
| d8     | 輕裝職業（盜賊、獵人、牧師） |
| d10    | 重裝職業（戰士、聖騎士、薩滿） |
| d12    | 蠻力職業（野蠻人） |

### 瀕死與死亡

1. HP 降至 0 → 進入**瀕死**狀態（倒地，無法行動）
2. 瀕死期間每回合開始擲**死亡豁免**：d20，10+ 成功，9- 失敗
3. 累計 3 次成功 → 穩定（HP = 1，意識恢復）
4. 累計 3 次失敗 → **死亡**
5. 自然 20 → 立即穩定並恢復 1 HP
6. 自然 1 → 算作 2 次失敗
7. 隊友可用**醫療檢定（DC 10）**或治療法術穩定瀕死者
8. 受到傷害 → 自動算 1 次失敗

### 角色死亡

- 死亡後可在城鎮復活（花費金幣，等級越高越貴）
- 復活費用 = 等級 x 100 金幣
- 如果金幣不足，可以接受「債務任務」來償還

### 升級獎勵

1. **HP 增加**：擲職業 HP 骰 + CON 調整值
2. **每 2 級**：獲得 1 個天賦點（見各戰役天賦樹）
3. **每 4 級（4、8、12、16、20）**：屬性提升 — 任選一個屬性 +2 或兩個屬性各 +1
4. **新技能**：按職業技能表解鎖新能力
5. **熟練加值提升**：按等級表自動提升

## 七、NPC 隊友系統

### 招募

- 在**城鎮或安全區域**可招募 NPC 隊友
- 隊伍上限：**4 人**（含玩家，最多 3 名 NPC）
- 每個 NPC 有招募條件（完成任務、聲望、花錢等）
- 招募時展示 NPC 資料（職業、性格、台詞預覽）
- 可以在城鎮解散隊友（他們會留在城鎮等待重新招募）

### AI 行為

NPC 隊友在戰鬥中由 DM 控制，遵循以下邏輯：

| 定位 | 行為優先級 |
|------|-----------|
| 坦克 | 攻擊距離最近的敵人 > 使用嘲諷技能 > 保護低 HP 隊友 |
| 治療 | 治療 HP < 50% 的隊友 > HP > 70% 時輸出 > 驅散減益 |
| 近戰 DPS | 攻擊 HP 最低的敵人 > 使用 AOE（3+ 敵人時）> 使用增益 |
| 遠程 DPS | 攻擊最危險的敵人（施法者優先）> AOE > 控制技能 |

### 玩家指令

玩家可以在自己的回合下達隊友指令：
- 「[名字]，治療我」→ NPC 下回合優先執行
- 「全體集火 [目標]」→ 所有 NPC 攻擊同一目標
- 「[名字]，使用 [技能]」→ NPC 下回合使用指定技能
- 「撤退」→ 所有 NPC 嘗試脫離戰鬥

### 探索中的表現

- NPC 會在特定場景觸發對話（進入其相關的區域、遇到特定敵人等）
- NPC 有好感度系統（隱藏數值），影響對話內容和特殊事件
- NPC 可能在危急時刻做出符合性格的自主行動

### 同步升級

- NPC 隊友與玩家**同等級**
- 玩家升級時 NPC 自動升級
- NPC 的技能和天賦由 DM 根據其性格自動選擇（偏向其定位最優解）
- NPC 裝備可由玩家分配（把掉落裝備給隊友）

## 八、物品與裝備

### 裝備槽位

| 槽位 | 數量 | 說明 |
|------|------|------|
| 武器 | 1（或雙持 2）| 決定攻擊方式和傷害 |
| 護甲 | 1 | 提供 AC 加值 |
| 盾牌 | 1 | 額外 AC +2（佔用一隻手） |
| 飾品 | 2 | 戒指、項鍊、披風等，提供特殊效果 |

### 背包

- 容量：20 格
- 每件物品佔 1 格（金幣不佔格）
- 滿了需要丟棄或出售

### 物品品質

| 顏色 | 品質 | 說明 |
|------|------|------|
| ⬜ 灰色 | 垃圾 | 只能賣錢 |
| ⬛ 白色 | 普通 | 基礎裝備，商店可買 |
| 🟩 綠色 | 優秀 | 有額外屬性加成 |
| 🟦 藍色 | 精良 | 副本 Boss 掉落，強力屬性 |
| 🟪 紫色 | 史詩 | 團本掉落，頂級裝備 |
| 🟧 橙色 | 傳說 | 極稀有，獨特效果 |

### 交易

- 購買：按物品標價
- 出售：物品標價的 **50%**
- NPC 商人在城鎮中，不同商人賣不同類型物品

## 九、休息與恢復

### 短休（10 分鐘）

- 在非戰鬥區域可進行
- 恢復 **25% 最大 HP**
- 每次副本限 **3 次**短休
- 不恢復消耗品

### 長休（在城鎮旅館）

- 花費金幣（等級 x 5g）
- **完全恢復** HP
- 恢復所有每日限制的技能
- 清除所有負面狀態

## 十一、副本規則

### 進入副本

- 部分副本需要前置任務
- 進入時 DM 讀取副本文件，展示概覽
- 建議隊伍配置提示（但不強制）

### 副本中保存

- 玩家可在**任何非戰鬥時刻**說「保存」
- Boss 戰中說「保存」→ 保存在 Boss 戰開始前的狀態
- 存檔記錄完整副本狀態（已清房間、已擊敗敵人、已獲物品、事件標記）

### 副本死亡

- 全隊陣亡 → 副本失敗
- 選項：
  1. 從上次保存點重新開始（恢復到保存時的狀態）
  2. 返回城鎮（副本進度重置，但保留經驗值和已獲物品）

### Boss 戰

- Boss 有多階段機制（HP 閾值觸發新技能）
- Boss 有戰前台詞和戰敗台詞
- Boss 掉落固定裝備（見副本掉落表）
- 擊敗 Boss 獲得大量 EXP

## 十二、隨機遭遇

在野外探索時，DM 可觸發隨機遭遇：

### 觸發規則

- 每次移動到新區域時，DM 掷 d20
- 結果 1-5：觸發遭遇（參考 enemies.md 的隨機遭遇表）
- 結果 6-10：發現環境事件（寶箱、NPC、線索）
- 結果 11-20：安全通過

### 遭遇類型

- 戰鬥遭遇：與區域等級匹配的敵人
- 商人遭遇：旅行商人，有稀有物品
- 事件遭遇：需要技能檢定的情境（救人、解謎、探索）
