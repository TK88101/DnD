# 血源戰役專屬規則

> 本文件包含「血源詛咒」戰役獨有的規則機制，與 `core.md` 配合使用。
> 本戰役不使用傳統護甲 AC 系統和治療職業，改用「閃避值」、「血瓶」和「洞察值」系統。
> 風格：暗黑、絕望、詭異、維多利亞哥特。一夜的狩獵，直到黎明。

---

## 一、血瓶系統

### 基礎規則

- 每場戰鬥開始時，獵人擁有 **10 瓶血瓶**
- 使用 1 瓶血瓶 = 恢復 **30% 最大 HP**（四捨五入）
- 使用血瓶消耗 **附贈動作**（不消耗主動作）
- 每場戰鬥限用 10 瓶，無法超額攜帶
- **城鎮補充**：回到獵人夢境或安全區域時，血瓶自動補滿
- 血瓶無法在戰鬥中分享給隊友

### 血瓶強化（通過升級獲得）

| 等級 | 效果 |
|------|------|
| Lv5 | 血瓶恢復量提升為 35% |
| Lv10 | 血瓶恢復量提升為 40% |
| Lv15 | 血瓶恢復量提升為 45% |
| Lv20 | 血瓶恢復量提升為 50% |

### 特殊血瓶

| 類型 | 效果 | 獲取 |
|------|------|------|
| 教會血瓶 | 恢復 50% HP + 解除中毒 | 教會獵人商店（30g） |
| 古老之血 | 恢復 40% HP + 攻擊 +2 持續 3 回合 | Boss 掉落 / 稀有 |
| 污穢之血 | 恢復 60% HP，但洞察值 +5 | 禁忌之森寶箱 |

## 二、閃避值系統（替代 AC）

### 基礎規則

本戰役的獵人不穿傳統重甲。防禦完全依賴閃避和直覺。

**閃避值 = 10 + DEX 調整值 + 獵人直覺**

### 獵人直覺

| 等級 | 獵人直覺加值 |
|------|------------|
| 1-4 | +0 |
| 5-8 | +1 |
| 9-12 | +2 |
| 13-16 | +3 |
| 17-20 | +4 |

### 獵人服裝（輕裝加值）

| 服裝 | 閃避加值 | 特殊效果 | 價格 |
|------|---------|---------|------|
| 普通衣物 | +0 | 無 | - |
| 獵人外套 | +1 | 毒素抗性 +1 | 50g |
| 教會長袍 | +1 | 神聖傷害抗性 10% | 80g |
| 亞南獵裝 | +2 | 出血抗性 +2 | 150g |
| 凱恩赫斯特血衣 | +2 | 暴擊率 +5% | 200g |
| 上位者之袍 | +3 | 洞察值相關檢定 +2 | 極稀有 |

### 閃避機制

- 每回合可使用 **1 次反應動作** 進行閃避：攻擊 DEX 豁免代替擊中閃避值
- **完美閃避**：DEX 豁免超過攻擊骰 5 以上，可立即移動 5 呎（不觸發機會攻擊）
- **翻滾**：消耗移動動作進行翻滾，該回合內閃避值 +2

## 三、內臟攻擊（處決）系統

### 觸發條件

當敵人處於 **「擊暈」** 狀態時，獵人可對其進行內臟攻擊。

### 內臟攻擊效果

- 傷害 = **武器傷害骰 x 3**（加所有傷害加值）
- 自動命中（不需攻擊骰）
- 恢復使用者 **15% 最大 HP**（獵人的血之回響）
- 消耗主動作

### 擊暈條件

| 方式 | 說明 |
|------|------|
| 連續命中 | 同一目標被連續命中 3 次（不同回合也可），目標擊暈 1 回合 |
| 暴擊 | 暴擊命中時，目標擊暈 1 回合 |
| 反擊（見下文） | 在敵人攻擊瞬間成功反擊，目標擊暈 2 回合 |
| 特定技能 | 某些職業技能可造成擊暈 |

### 擊暈免疫

- Boss 級敵人在被擊暈一次後，下次擊暈所需的連續命中提升為 5 次
- 某些敵人（如修格斯級別）完全免疫擊暈

## 四、反擊系統

### 機制

在敵人攻擊的瞬間，獵人可以使用 **反應動作** 進行槍擊反擊。

### 流程

1. 敵人宣告攻擊
2. 獵人宣告反擊（消耗反應動作 + 1 發子彈）
3. 獵人擲 DEX 檢定（DC = 10 + 敵人攻擊加值）
4. **成功**：敵人攻擊被打斷，進入擊暈狀態 **2 回合**（可處決）
5. **失敗**：反擊失敗，獵人本回合無法再閃避，敵人攻擊正常命中

### 反擊限制

- 每回合限 1 次
- 需要裝備遠程武器（獵人手槍）且有子彈
- 對「無法擊暈」的敵人仍可打斷攻擊，但不造成擊暈
- Boss 的某些特殊攻擊（標記為「不可打斷」）無法被反擊

## 五、洞察值系統

### 基礎規則

- 洞察值範圍：**0 - 99**
- 初始洞察值：**0**
- 洞察值代表獵人對世界「真相」的理解程度——看到的越多，世界就越扭曲

### 洞察值獲取

| 事件 | 洞察值變化 |
|------|----------|
| 首次遭遇新型敵人 | +1 |
| 擊殺 Boss | +10 |
| 閱讀禁忌知識 / 古籍 | +5 |
| 發現隱藏區域 | +3 |
| 使用古老之血 | +2 |
| 使用瘋人知識（物品） | +5 |
| 觸摸上位者的遺物 | +10 |

### 洞察值消耗

| 用途 | 洞察值消耗 |
|------|----------|
| 召喚其他獵人協助（多人機制） | -1 |
| 購買特殊物品 | -3 到 -10 |
| 使用某些強力技能 | -1 到 -5 |

### 洞察值階段效果

#### 第一階段：無知的幸福（0-20）

| 效果 | 描述 |
|------|------|
| 世界觀 | 正常的亞南城——哥特建築、瘋狂市民、野獸出沒 |
| 戰鬥 | 無特殊效果 |
| NPC | 正常互動 |
| 額外 | 無 |

#### 第二階段：啟蒙初現（21-40）

| 效果 | 描述 |
|------|------|
| 世界觀 | 開始看到牆壁上的隱藏符文、感知到建築中的秘密通道 |
| 戰鬥 | 新敵人「小蜘蛛」出現在特定區域；原本空曠的場景有了敵人 |
| NPC | 部分 NPC 對你態度改變（更恐懼或更親近） |
| 額外 | 感知隱藏物品和通道的被動檢定 +3 |
| 解鎖 | 獸化能力（見下文） |

#### 第三階段：真相初露（41-60）

| 效果 | 描述 |
|------|------|
| 世界觀 | 天空中開始出現巨大的、半透明的觸手；建築的影子不對 |
| 戰鬥 | 所有敵人 HP +10%；新的隱藏敵人「教會之眼」出現 |
| NPC | 更多 NPC 會給你禁忌的情報——但也有人開始迴避你 |
| 額外 | 所有洞察相關檢定 +2；獸化能力強化 |
| 代價 | 每次長休需 WIS DC 12 否則噩夢（恢復減半） |

#### 第四階段：深淵之眼（61-80）

| 效果 | 描述 |
|------|------|
| 世界觀 | 整個世界被血紅色的月亮照亮；遠處可以看到上位者的輪廓 |
| 戰鬥 | 所有敵人 HP +20%、攻擊 +2；上位者的使者開始出現 |
| NPC | 只有最瘋狂的 NPC 還願意和你說話 |
| 額外 | 獲得「眼球法術」（消耗洞察值的特殊能力，見下文）|
| 代價 | 每次長休 WIS DC 15 否則 HP 恢復減半；偶爾在非戰鬥場景出現幻覺 |

#### 第五階段：接近真相（81-99）

| 效果 | 描述 |
|------|------|
| 世界觀 | 你看到了一切。上位者無處不在。現實是一層薄膜。月亮不是月亮——它是一隻眼睛。 |
| 戰鬥 | 所有敵人 HP +30%、攻擊 +3；可以看到上位者並與之戰鬥 |
| NPC | 人偶是你唯一的交談對象 |
| 額外 | 所有能力達到頂峰；眼球法術免費使用 |
| 代價 | 每次長休自動噩夢（HP 恢復 25%）；WIS 永久 -1；但 INT +2 |

### 眼球法術（洞察值 61+ 解鎖）

| 法術 | 效果 | 洞察消耗 |
|------|------|---------|
| 奧秘之眼 | 看穿 30 呎內所有隱藏、幻覺和偽裝 | 3 |
| 觸手打擊 | 從手中射出奧術觸手，2d8 傷害，射程 30 呎 | 2 |
| 宇宙呼喚 | 30 呎範圍 3d8 奧術傷害 + 目標 WIS DC 16 否則恐懼 2 回合 | 5 |

## 六、獸化系統

### 解鎖條件

- 洞察值達到 **40** 以上
- 或使用特定物品「獸之擁抱」

### 獸化效果

| 屬性 | 變化 |
|------|------|
| STR | +4 |
| DEX | +4 |
| WIS | -3 |
| 持續 | 5 回合 |
| 外觀 | 獵人的外貌部分獸化——瞳孔變為獸瞳、指甲變為利爪、身體略微增大 |

### 獸化戰鬥

- 近戰攻擊自動使用利爪（2d6+STR）
- 無法使用槍械和物品（包括血瓶）
- 移動速度 x1.5
- 閃避值 +2
- 無法使用需要專注的技能

### 獸化結束

- 5 回合後自動解除
- 解除後 1 回合內所有檢定 -2（疲勞）
- 每場戰鬥限用 1 次
- **風險**：如果在獸化狀態下 HP 降至 10% 以下，需 WIS DC 15 否則失去控制（攻擊最近的生物，敵友不分）1d4 回合

## 七、武器變形系統

### 規則

- 每把變形武器都有 **兩種形態**：快速形態和重擊形態
- 戰鬥中切換形態消耗 **附贈動作**
- 切換形態時可進行一次 **變形攻擊**（兩種形態的傷害取平均值）

### 形態對比

| 形態 | 攻擊速度 | 傷害 | 特點 |
|------|---------|------|------|
| 快速形態 | 每回合 2 次攻擊 | 較低 | 靈活、適合反擊 |
| 重擊形態 | 每回合 1 次攻擊 | 較高 | 強力、適合處決 |

### 武器列表概覽

| 武器 | 快速形態 | 重擊形態 |
|------|---------|---------|
| 鋸肉刀 | 短刀（1d6+DEX，2 次） | 鋸齒刀（2d8+STR） |
| 獵人斧 | 手斧（1d6+STR，2 次） | 長柄斧（2d10+STR） |
| 路德維希聖劍 | 直劍（1d8+STR，2 次） | 巨劍（3d8+STR） |
| 教會連鎖 | 短劍（1d6+DEX，2 次） | 鏈鞭（2d6+DEX，射程 10 呎） |
| 刺劍 | 刺劍（1d8+DEX，2 次） | 長槍（2d8+DEX，射程 10 呎） |
| 千影 | 短刀（1d4+DEX，3 次） | 雙刀（1d6+DEX，3 次） |

> 完整武器數據見 `campaigns/bloodborne/items.md`

## 八、獵人手槍

### 規則

- 每個獵人裝備一把手槍作為副武器
- 手槍主要用途：**反擊**（見反擊系統）
- 也可作為遠程攻擊使用（傷害較低）
- 彈藥有限，需在安全區域補充

|------|------|------|------|------|

## 九、HP 回復規則

### 說明

本戰役沒有傳統的治療職業。HP 恢復主要依賴：

| 方式 | 效果 | 限制 |
|------|------|------|
| 血瓶 | 30-50% 最大 HP（隨等級提升） | 每場戰鬥 10 瓶 |
| 內臟攻擊 | 15% 最大 HP | 需擊暈敵人 |
| 激進回復 | 受傷後 3 秒內攻擊敵人，恢復損失 HP 的 20% | 限受傷的下一回合 |
| 長休 | 回到獵人夢境，恢復全部 HP | 需要安全區域 |
| 短休 | 在安全區域休息 1 小時，恢復 25% HP | 需要安全區域 |

### 激進回復（Rally 系統）

- 獵人受到傷害後，傷害量的 20% 暫存為「可恢復 HP」（顯示為灰色血條）
- 如果獵人在 **下一個回合** 攻擊命中敵人，恢復該灰色血條
- 如果獵人未在下一回合攻擊命中，灰色血條消失
- 這鼓勵積極進攻的戰鬥風格

## 十、死亡與復活

### 獵人之夢

- 獵人死亡後不會永久死亡（與克蘇魯不同）
- 死亡後獵人在 **獵人夢境** 中醒來
- **代價**：
  - 失去所有未消耗的血之回響（經驗/金幣）
  - 副本進度重置到最近的「燈火」（存檔點）
  - 如果在 Boss 戰中全滅，Boss 恢復全部 HP

### 燈火系統

- 每個副本區域有 1-3 個「燈火」——石燈柱
- 點亮燈火（需 1 分鐘，不可在戰鬥中）= 設置復活點
- 復活後回到最近的燈火，HP/血瓶全滿
- 但該區域的所有普通敵人也會重生（Boss 除外）

### 血之回響

- 擊殺敵人獲得「血之回響」= 經驗值和金幣
- 死亡時所有未使用的血之回響掉落在死亡地點
- 復活後可以回到死亡地點撿回（但如果再次死亡就永久消失）

## 十一、環境規則

### 夜晚階段

整個血源戰役發生在一個夜晚。夜晚隨主線推進分為三個階段：

| 階段 | 觸發 | 世界變化 |
|------|------|---------|
| 黃昏 | 戰役開始 | 天色昏暗，野獸活躍度中等 |
| 深夜 | 擊殺第 2 個 Boss | 月亮變紅，所有敵人強化 +10%，新區域開放 |
| 血月 | 擊殺第 4 個 Boss | 血紅的月亮覆蓋天空，所有敵人強化 +20%，上位者開始顯現 |

### 燃燒的亞南

- 市民在街上生火驅趕野獸
- 火堆附近 10 呎是安全區域（敵人不會靠近，除非被攻擊）
- 但燃燒的建築物可能坍塌（環境危害：2d6 火焰，DEX DC 13 閃避）
