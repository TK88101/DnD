# 🎲 D&D エンドレスアドベンチャー — マルチプレイヤーテキストRPG

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

ブラウザベースのマルチプレイヤー ダンジョンズ＆ドラゴンズ テキストRPGです。Google Gemini AIをダンジョンマスター（DM）として採用し、1〜8人が同時にプレイできます。

## 🎮 四つのキャンペーン

| キャンペーン | テーマ | 特徴 |
|------|------|------|
| ⚔️ アゼロス遠征 | ワールド・オブ・ウォークラフト風 | アライアンス/ホード陣営、12種族10職業、ダンジョン＆レイド |
| 🐙 霧の深淵 | クトゥルフ神話 | 1920年代の調査活動、正気度システム |
| 🩸 血月の狩り | ブラッドボーン風 | ゴシックホラー、変形武器 |
| 🐉 狩猟の刻 | モンスターハンター風 | 狩猟→素材→鍛冶、ネコタク機能、10種の武器 |

## ✨ 主な機能

- **マルチプレイヤーリアルタイム対戦** — WebSocket接続で1〜8人が同部屋でプレイ可能
- **AIダンジョンマスター** — Gemini 2.5 Flash による没入感あふれるナレーション
- **キャラクター作成システム** — 種族/職業/能力値の割り振り（MHは武器選択のみでプレイ開始可能）
- **コード制御バトルエンジン** — すべてのダイスロール/ダメージ/HPをサーバーコードで処理、Geminiはナレーションのみ
- **デュアル入力モード** — 番号選択またはフリーテキスト（Geminiが意図を解析→コードが実行）
- **MPマジックシステム** — 魔法職のリソース管理
- **9職業の完全スキル/タレントツリー** — 各職業11スキル（Lv1-20）＋3つのタレントツリー
- **召喚物AI** — インプ/ヴォイドウォーカー等が自動行動
- **動的難易度調整** — パーティ人数に応じてモンスターのステータスを自動スケーリング
- **外部メモリ** — ゲーム状態の永続化、会話履歴の自動トリミング、トークン消費を一定に維持
- **セーブ/ロード** — いつでも進行状況を保存し、別の部屋から再開可能
- **離席システム** — 60秒操作なしで自動AFK状態へ移行、NPCがキャラクターを引き継ぎ
- **退席プレイヤー処理** — ロード時に不在プレイヤーを自動検出し、キャラクターをNPC制御に切り替え
- **BGM背景音楽** — YouTubeによるシーン連動BGM自動切替（フォールバックあり）
- **ボスキー** — Escキーで偽装画面に切り替え
- **キャラクターカラーシステム** — 各キャラクターに固有カラー、ナレーションは白、選択肢はシアン

### モンスターハンター専用機能
- 🐱 **ネコタク機能** — 力尽き3回でクエスト失敗、モンスターのHPは回復しない
- ⚔️ **部位破壊** — 頭/尾/翼を破壊するとモンスターの能力に影響
- 🎯 **捕獲システム** — 瀕死状態→罠→麻酔玉で捕獲、捕獲報酬はより豊富
- 🛒 **コード制御ショップ** — 雑貨屋はサーバーのコードで処理し、AIを経由しない
- 📈 **コード制御レベルアップ** — EXP到達で自動レベルアップ、コードがダイスロールでHPを計算

## 🚀 クイックスタート

### 動作環境
- Node.js 18+
- Google Gemini API Key

### インストール

```bash
git clone https://github.com/あなたのユーザー名/DnD.git
cd DnD/server
npm install
```

### 起動

```bash
export GEMINI_API_KEY="あなたのAPIキー"
node relay.js
```

ブラウザで `http://localhost:8080` を開いてください。

### リモートプレイ（Cloudflare Tunnel）

```bash
cloudflared tunnel --url http://localhost:8080
```

生成されたURLを友人に共有すれば参加できます。

## 🎮 ゲームコマンド

ゲーム内コマンドは中国語のまま入力します（そのまま動作します）。

| コマンド | 説明 |
|------|------|
| `開始遊戲` | 新しいゲームを開始 |
| `讀取 名前` / `讀檔 名前` | セーブデータを読み込む |
| `保存` / `存檔` | ゲームを保存 |
| `結束遊戲` / `退出` | 保存して終了 |
| `雜貨店` / `/shop` | ショップを開く（コード制御） |
| `/fight` | ランダム戦闘をトリガー（コード制御） |
| `/back` / `回來了` | 離席状態を解除 |
| `Esc` | ボスキー（偽装画面に切り替え） |

## 🏗️ 技術的特徴

- **Gemini 非信頼原則** — レベルアップ、ショップ、難易度、戦闘判定はすべてコードで制御、Geminiはナレーションのみ
- **バトルエンジン** — ダイス/ダメージ/HP/ドロップをコード処理、Geminiは意図解析とナレーション生成（2回のAPI呼び出し）
- **選択肢変換** — 数字入力→選択肢参照、フリーテキスト→Gemini意図解析→コード実行
- **外部メモリ** — AIの返答からステータス（HP/ゴールド/アイテム）を解析し、次のメッセージに注入
- **動的難易度** — プレイヤー人数に応じてHP倍率と攻撃補正をコードで計算
- **ハートビート維持** — 15秒ごとにpingを送信し、Cloudflareの90秒タイムアウト切断を防止
- **切断再接続** — 5分間部屋を保持し、ホスト/プレイヤーが再接続可能
